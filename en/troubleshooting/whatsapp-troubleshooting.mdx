> Common issues and solutions for WhatsApp Business integration

## Sender Issues

### Sender Stuck in "Creating" or "Pending"

**Symptoms:**

- Sender remains in Creating/Pending for too long
- Meta signup completed but sender never activates

**Solutions:**

1. Complete full Meta signup flow
2. Create a new WhatsApp Business account during setup
3. Verify correct number selection
4. Wait a few minutes and refresh
5. Contact support if status remains stuck for 30+ minutes

### Verification Failed

**Solutions:**

1. Wait 5-10 minutes before retrying
2. Try a different number
3. Use a normal mobile number type

### Sender Offline or Suspended

**Solutions:**

1. Review policy/quality notifications from Meta
2. Reduce spam-like messaging behavior
3. Re-authenticate connection if needed
4. Appeal through Meta if suspended

## Template Issues

### Template Rejected

Common fixes:

- Use correct template category
- Add realistic variable samples
- Avoid URL shorteners
- Remove aggressive language

### Template Pending Too Long

- Marketing templates can take up to 48 hours
- Create an alternate template if urgent

### Template Disabled

- Usually caused by poor quality feedback
- Create improved version and narrow audience targeting

## Messaging Issues

### Messages Not Delivering

1. Use E.164 number format (`+14155551234`)
2. Confirm recipient has WhatsApp
3. Confirm sender status is Online
4. Verify messaging limits are not exceeded

### Free-form Messages Failing

- Free-form messages work only within 24 hours of customer's last message
- Use approved template outside that window

### AI Not Responding

1. Verify assistant is connected to sender
2. Confirm AI is enabled
3. Check conversations/logs for processing errors

## Quality and Limits

### Quality Rating Dropped

- Review recent campaigns for block/report triggers
- Improve targeting and relevance
- Reduce sending volume until quality recovers

### Hitting Message Limits

- Limits reset on rolling windows
- Improve quality to unlock higher tiers
- Spread sends over time

## Connection Issues

### Meta Popup Doesn't Appear

- Allow popups
- Clear cache/cookies
- Retry in another browser

### Redirected Back Without Completion

- Re-run flow and finish all Meta steps
- Check for popup-side errors

## Getting Help

Provide support with:

- Sender ID
- Exact error text
- Steps already attempted
- Screenshots
