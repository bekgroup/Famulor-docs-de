# Sipgate Limitations

<Note>
  Unfortunately, Sipgate currently does not support static SIP trunks for integration with Famulor.
</Note>

## Why doesn’t the integration work right now?

<Card
  title="Current Technical Limitation"
  icon="triangle-exclamation"
  color="red">
  **Unfortunately, Sipgate currently does not support static SIP trunks.**
  
  This means that a direct SIP integration between Sipgate and Famulor is technically not possible at this time.
</Card>

<Card
  title="Ask the Provider Directly"
  icon="phone"
  color="blue">
  **Important:** Please ask Sipgate directly if they can offer static SIP trunks!
  
  
    - Many providers continuously develop their services
    - The feature may already be available or planned
    - Direct inquiry can lead to customized solutions
  
</Card>
