# Import a Fonial Phone Number into Famulor

This guide walks you through importing a Fonial number into Famulor.  
The process has three parts: create the number in Famulor, configure forwarding in Fonial, and complete the import.

## Prerequisites

- An active Fonial account with at least one phone number
- A Famulor account
- Access to the Fonial customer portal at [kundenkonto.fonial.de](https://kundenkonto.fonial.de)

---

## Step 1: Identify the number in Fonial

1. Sign in to your Fonial customer account.
2. Go to **Telefonanlage -> Rufnummern**.
3. Find the number you want to import into Famulor.
4. Note the number in **E.164 format**, for example `+498956546546`.

---

## Step 2: Create the number in Famulor

1. Open Famulor at [app.famulor.de/phone-numbers?lang=en](https://app.famulor.de/phone-numbers?lang=en).
2. In the sidebar, go to **Your phone numbers**.
3. Click **+ Integrate SIP trunk**.
4. Enter your SIP data:
   - **SIP extension:** your Fonial phone number (E.164 format)
   - **Username** and **password:** credentials provided by Fonial
   - **SIP address:** `sip.plusnet.de`
5. Under **Incoming call settings**, copy **Our SIP address**.
6. Click **Add SIP number**.

<Note>
  You need the copied **Our SIP address** in the next step as the target address in Fonial.
</Note>

<Note>
  The origination URL format is: `sip:49xxxxxxxxxxxx@xxxxxx.eu.sip.livekit.cloud`  
  Important: the phone number in this URL must be entered **without the plus sign (`+`)**.
</Note>

---

## Step 3: Create a forwarding target in Fonial

1. Go back to the Fonial customer account.
2. Navigate to **Telefonanlage -> Ziele**.
3. Open the **Weiterleitungen** tab.
4. Click **Neues Ziel anlegen**.
5. Choose **Weiterleitung (Mobilfunk, Festnetz, Fax oder SIP-URI)** as the type.
6. Fill in the fields like this:

| Field | Value |
| --- | --- |
| Forwarding target name | Any name, e.g. `Famulor` |
| Forwarding type | **Forward to SIP-URI** |
| SIP-URI | Paste the origination URL copied from Famulor |
| Outgoing phone number | Select the Fonial number you import |
| Outgoing caller ID display | Select your preferred display option |

7. Click **Save**.

---

## Step 4: Activate call forwarding on the number

1. In Fonial, go to **Telefonanlage -> Rufnummern**.
2. Click the **Edit** icon (pencil) for the target number.
3. Click **Neue Weiterleitung anlegen**.
4. Under **Forwarding target**, select:
   - **Forward to:** the target you created (e.g. `Famulor`)
   - **for:** `Unbegrenzt` (unlimited)
5. Click **Weiterleitung speichern**.
6. Then click **Speichern** at the bottom to apply the full number configuration.

No additional time rules or conditions are required.  
With `Unbegrenzt`, all incoming calls are directly forwarded to Famulor.

---

## Step 5: Assign an assistant

To have incoming calls answered by your AI assistant, assign the imported number to an assistant.

1. Open **Assistants** in Famulor.
2. Edit the assistant you want to use.
3. Select the correct **inbound type**.
4. Choose the imported phone number from the list.
5. Click **Save assistant**.
6. Place a test call.
