## What is Zadarma Integration?

Zadarma integration allows you to connect your Zadarma phone system with Famulor. Zadarma offers flexible SIP Trunking solutions for telephony integration.

<Steps>
  <Step title="Use Existing Numbers">
    Use your existing Zadarma phone numbers with our AI assistants
  </Step>
  <Step title="Maintain Infrastructure">
    Keep Zadarma’s robust infrastructure and global reach
  </Step>
  <Step title="Manage Calls">
    Make and receive calls through your Zadarma setup
  </Step>
  <Step title="Retain Control">
    Maintain full control over your numbers
  </Step>
  <Step title="Save Costs">

  </Step>
</Steps>

## Setting Up Your Zadarma Integration

<Steps>
  <Step title="Gather Information">
    <CardGroup cols={2}>
      <Card
        title="PBX Extension"
        icon="phone"
        color="blue">
        **Navigate to "My PBX" → "Extensions" on the main page.**
        
        ![Zadarma My PBX Extensions](/images/provisioning/sip-ai/zadarma-mypbx-extensions.png)
        
        Locate the following information:
        <CodeGroup>
          <CodeGroupItem title="Example">
            ```bash
            Extension Number: e.g. 1000
            Server Address: e.g. pbx.zadarma.com
            Username: Your extension username
            Password: Your extension password
            ```
          </CodeGroupItem>
        </CodeGroup>
        
        ![Zadarma Extension Settings](/images/provisioning/sip-ai/zadarma-switch-extension-settings.PNG)
      </Card>
      <Card
        title="SIP Connection"
        icon="server"
        color="green">
        **Go to "Configuration" → "SIP Settings" on the main page.**
        
        ![Zadarma SIP Settings](/images/provisioning/sip-ai/zadarma-sip-settings.png)
        
        Find the following information:
        <CodeGroup>
          <CodeGroupItem title="Example">
            ```bash
            Phone Number: e.g. +1 555 0123
            SIP Server: e.g. sip.zadarma.com
            Username: Provided by your phone provider
            Password: Provided by your phone provider
            ```
          </CodeGroupItem>
        </CodeGroup>
        
        ![Zadarma SIP Connection Settings](/images/provisioning/sip-ai/zadarma-sip-connection-settings.png)
      </Card>
    </CardGroup>
    
    <Card
      title="IP Authorization Type"
      icon="shield"
      color="orange">
      **For IP Authorization Type:**
      
      1. Go to "Services" → "Connect Number" on the main page
      
      ![Zadarma Services](/images/provisioning/sip-ai/zadarma-services.png)
      
      2. Open "Active Numbers"
      
      ![Zadarma Active Numbers](/images/provisioning/sip-ai/zadarma-active-numbers.png)
      3. Click the number you want to connect and press the settings button
      
      ![Zadarma Settings Button](/images/provisioning/sip-ai/zadarma-settings-button.png)
      
      4. Click on "External Services" then enable the SIP URI option
      
      ![Zadarma External Services](/images/provisioning/sip-ai/zadarma-external-services.png)
      5. Add `sip:extension@your-unique-id.sip.livekit.cloud` (this is the Famulor SIP server address)
      
      <Note>
        You will need these IPs, which you enter in Step 2 for the incoming authorization type.
      </Note>
    </Card>
    
    <Note>
      **Important:** The specific Famulor SIP address is available in your Famulor dashboard under "Phone Numbers" → "Integrate SIP Trunk" → "Our SIP Address."
    </Note>
  </Step>
  
  <Step title="Import Numbers to Famulor">
    <Card
      title="Import Process"
      icon="download"
      color="blue">
      You need to import them into Famulor.
      
      1. Go to your Famulor dashboard
      2. Navigate to "Your Phone Numbers"
      3. Click on "Integrate SIP Trunk"
    </Card>
    
    <CardGroup cols={2}>
      <Card
        title="PBX Extension Details"
        icon="gear"
        color="purple">
        Fill in your details based on your integration type.
      </Card>
      <Card
        title="SIP Connection Details"
        icon="link"
        color="green">
        Fill in your details based on your integration type.
      </Card>
    </CardGroup>
    
    <Card
      title="Server Address"
      icon="server"
      color="blue">
      Enter the termination SIP URI that you configured in Step 1 here.
      
      <CodeGroup>
        <CodeGroupItem title="Example">
          ```bash
          Server Address: e.g. pbx.zadarma.com / sip.zadarma.com depending on your integration type
          ```
        </CodeGroupItem>
      </CodeGroup>
    </Card>
    
    <Card
      title="Authorization Type"
      icon="shield"
      color="orange">
      Select the authorization type here.
      
      Add the IPs from Zadarma (Step 1).
    </Card>
    
    <Card
      title="Select Country"
      icon="globe"
      color="green">
      Choose the country where your SIP trunk is located.
    </Card>
  </Step>
  
  <Step title="Verify Setup">
    <Card
      title="Verification Steps"
      icon="check-circle"
      color="green">
      After configuration:
      
      1. Make a test call to your number/extension
      2. Verify that the AI assistant answers
      3. Use the test call feature to verify outgoing calls
    </Card>
  </Step>
</Steps>

## Security Requirements

<Card
  title="SIP Credentials Security"
  icon="shield"
  color="red">
  Your SIP credentials must be secure:
  
  - **Username:** Your extension number or the username provided by Zadarma
  - **Password requirements:**
    - 12 or more characters
    - Numbers
    - Upper and lowercase letters
    - Special characters recommended
</Card>

## Common Issues

<CardGroup cols={2}>
  <Card
    title="Outgoing Calls Not Working"
    icon="phone-slash"
    color="red">
    **Incoming calls work after connection, but outgoing calls don’t?**
    
    <Tabs>
      <TabItem title="Termination SIP URI">
        - Remove any spaces if present
        - Use a localized termination URI close to your region
        - Refer to this documentation to choose one
      </TabItem>
      <TabItem title="Username & Credentials">
        - Make sure you entered the correct username and credentials
        - Note that the username is not the friendly name shown in the credentials list
        - The friendly name differs from the username - double-check this
      </TabItem>
    </Tabs>
  </Card>
  
  <Card
    title="Setting up International Countries"
    icon="globe"
    color="blue">
    **How do I set up dialing to international countries?**
    
    1. Search for "geo" to find the "Voice Geographic Permissions" setting
    2. Select "Elastic SIP Trunking" in the selector
    3. Choose the countries you want to dial
  </Card>
</CardGroup>
