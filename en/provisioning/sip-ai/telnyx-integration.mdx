## What is Telnyx Integration?

Telnyx integration allows you to connect your Telnyx phone system with Famulor. Telnyx provides flexible SIP Trunking solutions with FQDN settings for telephony integration.

<Steps>
  <Step title="Create SIP Trunk">
    Create a SIP Trunk with FQDN settings in Telnyx
  </Step>
  <Step title="Adjust Configuration">
    Adjust the inbound and outbound settings
  </Step>
  <Step title="Add Number">
    Add your purchased number to the trunk
  </Step>
  <Step title="Configure Famulor">
    Create the number in Famulor using the Telnyx settings
  </Step>
  <Step title="Assign Assistant">
    Assign the number to an AI assistant
  </Step>
</Steps>

## Setting Up Your Telnyx Integration

<Steps>
  <Step title="Create SIP Trunk">
    <Card
      title="FQDN Setting"
      icon="plus-circle"
      color="blue">
      **Create a SIP Trunk with FQDN setting:**
      
        - Add <code>your-unique-id.sip.livekit.cloud</code>
        
        <Note>
          **Important:** You can find the specific Famulor SIP address in your Famulor dashboard under "Phone Numbers" → "Integrate SIP Trunk" → "Our SIP Address".
        </Note>
        - Select **SRV** as the record type
      
      <Note>
        The FQDN setting is crucial for correctly routing calls to Famulor.
      </Note>
    </Card>
    
    <Card
      title="Credentials (Optional for outgoing calls)"
      icon="key"
      color="green">
      **Add credentials for outgoing calls:**
      
      The username and password can be chosen freely.
      
      <Note>
        This setting is optional and only necessary if you plan to use outgoing calls.
      </Note>
    </Card>
  </Step>
  
  <Step title="Inbound Settings">
    <Card
      title="Inbound Configuration"
      icon="gear"
      color="purple">
      **Configure the inbound settings:**
      
        - **Number Format:** Change the number format to <code>+E.164</code> (note: don’t forget the +)
        - **Protocol:** Select **TCP** as the protocol
        - **Region:** Set the region to **Europe**
        - **Codecs:** Restrict codecs to those selected in the screenshot
      
      <Note>
        The +E.164 format is important for correct transmission of international numbers.
      </Note>
    </Card>
  </Step>
  
  <Step title="Outbound Settings">
    <Card
      title="Outbound Settings V2"
      icon="phone"
      color="orange">
      **Configure the outbound settings:**
      
        - **Profile:** Choose **Default** as the profile
        - **Country:** Select your country
      
    </Card>
  </Step>
  
  <Step title="Add Number to Trunk">
    <Card
      title="Add Number"
      icon="phone"
      color="green">
      **Finally, add the purchased number to the trunk.**
      
      Ensure the number is correctly assigned to the created SIP Trunk.
    </Card>
  </Step>
  
  <Step title="Create Number in Famulor">
    <Card
      title="Famulor Configuration"
      icon="download"
      color="blue">
      **Create the number in Famulor:**
      
        - **Number / Name:** Enter exactly the number you added
        - **Optional for Outbound:**
          
            - **Termination URL:** <code>sip.telnyx.eu</code>
            - **Username:** Username from the configuration
            - **Password:** Password from the configuration
        
    </Card>
  </Step>
  
  <Step title="Assign Assistant">
    <Card
      title="Assignment"
      icon="check-circle"
      color="purple">
      **Now you can save the number and assign it to an assistant.**
      
      The integration is complete and the number can be used for incoming and outgoing calls.
    </Card>
  </Step>
</Steps>

## Security Requirements

<Card
  title="SIP Credentials Security"
  icon="shield"
  color="red">
  Your SIP credentials must be secure:
  
    - **Username:** Your Telnyx username or the provided username
    - **Password requirements:**
      
        - 12 or more characters
        - Numbers
        - Uppercase and lowercase letters
        - Special characters recommended
      
</Card>

## Common Issues

<CardGroup cols={2}>
  <Card
    title="FQDN Configuration"
    icon="circle-exclamation"
    color="red">
    **FQDN is not recognized correctly?**
    
      - Make sure <code>xxx.eu.sip.livekit.cloud</code> is entered correctly
      - Select **SRV** as the record type
      - Check for typos
    
  </Card>
  
  <Card
    title="E.164 Format Issues"
    icon="phone-slash"
    color="orange">
    **Calls are not routed correctly?**
    
      - Verify the <code>+E.164</code> format is chosen
      - Ensure the "+" is not missing
      - Test with a simple phone number
    
  </Card>
  
  <Card
    title="TCP Protocol"
    icon="server"
    color="blue">
    **Connection problems?**
    
      - Confirm TCP is selected as the protocol
      - Check your firewall settings
      - Contact Telnyx support if issues persist
    
  </Card>
  
  <Card
    title="Codec Issues"
    icon="gear"
    color="purple">
    **Audio quality problems?**
    
      - Check that recommended codecs are selected
      - Ensure codec restriction is enabled
      - Test with different codec settings
    
  </Card>
</CardGroup>
