# Vonage Integration

<Note>
  A step-by-step guide to connecting your Vonage account via SIP trunking with Famulor.
</Note>

## Overview

<Card
  title="What is Vonage?"
  icon="building"
  color="blue">
  Vonage is a leading provider of cloud communication solutions. With the SIP trunking integration, you can connect your existing Vonage phone numbers with Famulor and benefit from AI-powered call management.
</Card>

## Step 1: Create Elastic SIP Trunking

<Card
  title="Set up SIP trunk in Vonage"
  icon="gear"
  color="green">
  
    - Navigate to the **SIP section** in the Vonage dashboard
    - Select **"Other"** as provider
    - Follow the step-by-step instructions
  
</Card>

### Configure Termination (for outgoing calls)

<Card
  title="Note important information"
  icon="edit"
  color="orange">
  When setting up termination (outgoing calls), the following information is important:
  
    - **Termination SIP URI**
    - **Username**
    - **Password**
  
  **Important:** Note down this information as you will need it in later steps.
</Card>

### Configure Origination (for incoming calls)

<Card
  title="Configure Famulor SIP server"
  icon="server"
  color="purple">
  Here, enter the Famulor SIP server address as the Origination SIP URI:
  <CodeGroup>
    <CodeGroupItem title="SIP Server">
      ```bash
      sip:your-unique-id.sip.livekit.cloud
      ```
    </CodeGroupItem>
  </CodeGroup>
  
  <Note>
    **Important:** The specific Famulor SIP address can be found in your Famulor dashboard under "Phone Numbers" → "Integrate SIP Trunk" → "Our SIP Address".
  </Note>
</Card>

## Step 2: Move numbers to Elastic SIP Trunking

<Card
  title="Link numbers"
  icon="phone"
  color="blue">
  After creating the Elastic SIP trunk, you need to:
  
    - Purchase new numbers **OR**
    - Link existing numbers to this trunk
  
</Card>

## Step 3: Import numbers into Famulor

<Card
  title="Complete integration in Famulor"
  icon="download"
  color="green">
  Now that the number is configured with your Elastic SIP trunking, you need to import it into Famulor so that we know how to route the calls.
</Card>

### Required information

<Card
  title="Enter Vonage data"
  icon="database"
  color="red">
  Enter the following information you set up in Step 1:
  
    - **Termination SIP URI** from Step 1
    - **Username** from Step 1
    - **Password** from Step 1
  
</Card>

### Programmatic import

<Card
  title="API import available"
  icon="code"
  color="indigo">
  You can also import numbers programmatically via the **Import Number API**.
</Card>

## Common issues and solutions

<Card
  title="Troubleshooting"
  icon="circle-question"
  color="orange">
  
    - **Connection issues:** Check the SIP URI and login credentials
    - **Calls not working:** Make sure the trunk is enabled
    - **Missing incoming calls:** Verify the origination settings
    - **Missing outgoing calls:** Verify the termination settings
  
</Card>

## Setup in Famulor

<Card
  title="Setup steps"
  icon="gear"
  color="purple">
  
    - Go to **Phone Numbers** → **Integrate SIP Trunk**
    - Select **"Vonage"** as provider
    - Enter the **Termination SIP URI**
    - Enter the **Username**
    - Enter the **Password**
    - Click **"Save"**
    - Test the connection by placing a call
  
</Card>
