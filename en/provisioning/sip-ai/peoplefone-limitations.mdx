# Peoplefone Limitations

<Note>
  Unfortunately, Peoplefone does not currently support static SIP trunks for integration with Famulor.
</Note>

## Why does the integration currently not work?

<Card
  title="Current Technical Limitation"
  icon="triangle-exclamation"
  color="red">
  **Peoplefone unfortunately does not currently support static SIP trunks.**
  
  This means that a direct SIP integration between Peoplefone and Famulor is technically not possible at this time.
</Card>

<Card
  title="Check Directly with the Provider"
  icon="phone"
  color="blue">
  **Important:** Please ask Peoplefone directly whether they can offer static SIP trunks!
  
  
    - Many providers continuously develop their services
    - The feature may already be available or planned
    - Direct inquiry can lead to individual solutions
  
</Card>

## What does this mean for you?

<CardGroup cols={2}>
  <Card
    title="Limited Functionality"
    icon="phone-slash"
    color="orange">
    
      - No direct SIP trunk integration possible
      - No automatic forwarding of calls to AI assistants
      - Limited automation options
    
  </Card>
  
  <Card
    title="Alternative Solutions"
    icon="lightbulb"

      - Use of Famulor numbers

  </Card>
</CardGroup>

## Our Recommendation

<Card
  title="Simple Alternative"
  icon="star"
  color="green">
  **You can purchase and use a number directly through us!**
  
  
    - **Inbound:** Simply forward your current Peoplefone number to the Famulor number
    - **Outbound:** Verify the number under "Phone numbers" and place calls directly with it
  
  
  This solution provides you with:
  
  
    - Full AI assistant functionality
    - Automated call handling
    - Advanced telephony features
    - Seamless integration with the Famulor platform
    - Easy setup via call forwarding
  
</Card>
