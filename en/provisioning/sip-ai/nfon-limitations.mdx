# NFON Limitations

<Note>
  Unfortunately, NFON currently does not support static SIP trunks for integration with Famulor.
</Note>

## Why doesn't the integration work at the moment?

<Card title="Current Technical Limitation" icon="question" color="red">
  Unfortunately, NFON currently does not support static SIP trunks.

  This means that a direct SIP integration between NFON and Famulor is technically not possible at this time.
</Card>

<Card title="Inquire Directly with the Provider" icon="phone" color="blue">
  Important:

   Please ask NFON directly if they can offer static SIP trunks\!

  
  - 
  Many providers are continuously developing their services

  
  - 
  The feature might already be available or planned

  
  - 
  Direct inquiries can lead to customized solutions

  
</Card>
