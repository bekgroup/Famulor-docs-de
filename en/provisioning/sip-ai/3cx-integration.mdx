## What is 3CX Integration?

3CX Integration allows you to connect your 3CX phone system with Famulor. 3CX is a popular IP PBX that provides flexible SIP trunking solutions for telephony integration.

<Steps>
  <Step title="Utilize Existing Infrastructure">
    Connect your existing 3CX phone system with our AI assistants.
  </Step>
  <Step title="Simple Configuration">
    Easy setup via the 3CX admin interface.
  </Step>
  <Step title="Manage Calls">
    Make and receive calls through your 3CX setup.
  </Step>
  <Step title="Maintain Control">
    Retain full control over your 3CX configuration.
  </Step>
  <Step title="Save Costs">
    Benefit from your existing 3CX infrastructure.
  </Step>
</Steps>

## Setting Up Your 3CX Integration

<Steps>
  <Step title="Create a Trunk in 3CX">
    <Card
      title="Trunk Creation"
      icon="plus-circle"
      color="blue">
      **Navigate to "Trunks & Chat" in the admin interface and create a new trunk using the +Trunk button.**
      <Note>
        Select the option "Generic SIP Trunk (IP Based)". **Note - This is only available for hosted systems!**
      </Note>
    </Card>
  </Step>
  <Step title="Configure 3CX Trunk">
    <CardGroup cols={2}>
      <Card
        title="Basic Configuration"
        icon="gear"
        color="green">
        
          - **Choose any name you like**
          - **Use the number** (including extension) that you want to be reachable through Famulor
          - **Set Registrar** to <code>your-unique-id.sip.livekit.cloud</code>
        
        <Note>
          **Important:** You can find your specific Famulor SIP address in your Famulor dashboard under "Phone Numbers" → "Integrate SIP Trunk" → "Our SIP Address".
        </Note>
      </Card>
      <Card
        title="Transport Protocol"
        icon="shield"
        color="orange">
        **Restrict the transport protocol to TCP.**
        This is important for maintaining a stable connection between 3CX and Famulor.
      </Card>
    </CardGroup>
  </Step>
  <Step title="Create Number in Famulor">
    <Card
      title="Number in Famulor"
      icon="phone"
      color="purple">
      **Go to Phone Numbers - Add Number - Create SIP Number and add the number in Famulor.**
      <Note>
        Use exactly the same number you configured above.
      </Note>
    </Card>
  </Step>
  <Step title="Outgoing Calls">
    <Card
      title="Functionality"
      icon="phone"
      color="blue">
      **Outgoing calls - Coming soon!**
      This feature will be available shortly.
    </Card>
  </Step>
  <Step title="Use the Trunk">
    <Card
      title="Trunk Usage"
      icon="check-circle"
      color="green">
      **You can now set this trunk in 3CX and use it wherever you want.**
      The trunk is now configured and can be used for incoming calls.
    </Card>
  </Step>
</Steps>

## Important Notes

<CardGroup cols={2}>
  <Card
    title="Hosted Systems Required"
    icon="triangle-exclamation"
    color="red">
    **Important:** The "Generic SIP Trunk (IP Based)" option is only available on hosted 3CX systems.
    Local installations may not support this feature.
  </Card>
  <Card
    title="TCP Protocol"
    icon="shield"
    color="orange">
    **Transport Protocol:** The transport protocol must be restricted to TCP for a stable connection.
  </Card>
  <Card
    title="Number Matching"
    icon="phone"
    color="blue">
    **Number Matching:** The number in 3CX must exactly match the number in Famulor.
  </Card>
  <Card
    title="Registrar Address"
    icon="server"
    color="green">
    **Registrar:** Use the correct Famulor SIP server address.
  </Card>
</CardGroup>

## Security Requirements

<Card
  title="SIP Credentials Security"
  icon="shield"
  color="red">
  Your SIP credentials must be secure:

        - Numbers
        - Upper and lower case letters
        - Special characters recommended

## Common Issues

<CardGroup cols={2}>
  <Card
    title="Trunk Option Not Available"
    icon="circle-exclamation"
    color="red">
    **The "Generic SIP Trunk (IP Based)" option is not available?**
    
      - Check if you are using a hosted 3CX system.
      - Local installations may not support this feature.
      - Contact 3CX support for more information.
  </Card>
  <Card
    title="Connection Problems"
    icon="phone-slash"
    color="orange">
    **Connection problems between 3CX and Famulor?**
    
      - Ensure the transport protocol is set to TCP.
      - Verify the registrar address is correct.
      - Check your firewall settings.
  </Card>
  <Card
    title="Number Mismatch"
    icon="phone"
    color="blue">
    **Calls are not being forwarded correctly?**
    
      - Make sure the number in 3CX exactly matches the number in Famulor.
      - Check your extension configuration.
      - Test with a simple call.
  </Card>
  <Card
    title="Outgoing Calls"
    icon="phone"
    color="purple">
    **Outgoing calls not working?**
    <Note>
      Outgoing calls are currently not available. This functionality will be added soon.
    </Note>
  </Card>
</CardGroup>
