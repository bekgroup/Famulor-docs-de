# Trello Board Templates

Use these template patterns to roll out repeatable Trello workflows quickly across teams.

## Standard Templates

- Sales Pipeline: Lead -> Qualified -> Proposal -> Won
- Support Workflow: New -> In Progress -> Waiting -> Resolved
- Product Board: Ideas -> Backlog -> Development -> Review -> Done

## Template Rules

1. Use consistent label naming per team.
2. Enforce owner and due date as required fields.
3. Define archive rules for completed cards.
