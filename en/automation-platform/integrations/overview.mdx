# Famulor Automation - Integrations Overview

Famulor Automation is a comprehensive no-code automation platform that connects your AI phone assistants with over **379 external applications and services**. The platform allows businesses to create advanced call automation workflows without writing code, integrating voice interactions with external systems to optimize processes and enhance customer experiences.

## Core Automation Triggers

### Call-Based Triggers

#### Call Ended Trigger
- Activates immediately when an AI call ends
- Provides access to full call transcription and AI responses
- Includes call duration, status, and detailed metadata
- Enables post-call automation such as CRM updates and follow-up actions

#### Incoming Call Variable Injection
- Activates before the AI assistant answers an incoming call
- Enriches AI context with real-time customer data
- Adjusts call flow based on caller information
- Enables personalized interactions from the very first moment
