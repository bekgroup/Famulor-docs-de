Famulor includes built-in **appointment scheduling** so your voice assistant can work with common online calendars. You enable it per assistant under **Prompt & Tools** in the **Standard tools** section. Besides **Cal.com**, you can choose **GoHighLevel** or **Calendly** — the overall flow is the same.

## Where to configure it

1. Open your assistant → **Prompt & Tools**.
2. Scroll to **Standard tools** and select **Appointment scheduling** (scheduling / booking).
3. Pick your provider: **Cal.com**, **GoHighLevel**, or **Calendly**.

## What to set first: when to book

Define **when** the assistant should offer or complete a booking. Use clear instructions (you can paste a polished prompt into the tool’s description field). Examples: after qualification, when the caller asks for a viewing, or when a specific service is chosen — match this to your industry template (real estate, onboarding call, etc.).

## Cal.com: connect with an API key

1. Log in to [Cal.com](https://cal.com) and create the **event types** you want (for example “Property interest — Zoom” or “Onboarding meeting”).
2. In Cal.com, open **Settings**, scroll to **Developer**, then **API keys**.
3. Click **Add**, name the key, set **expiration** as needed (for testing, “never expires” is simple), then **save**.
4. **Copy** the API key and paste it into the Famulor assistant’s Cal.com field.
5. If the connection works, the **event** list loads in Famulor — select the right event and **refresh / update fields** so booking fields match.

## Multiple calendars or services

Use **Add** / **+** in the Famulor tool UI to attach **another** calendar connection if you run several offerings (for example different treatments or meeting types). For each entry, describe **when** booking should use that calendar.
