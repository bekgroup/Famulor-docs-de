## 1. Create a new custom dashboard

1. In your account, click **Create Custom Dashboard**.
2. Enter a name for the dashboard.
3. Choose a layout style (for example squares or table view).
4. Create the dashboard.

## 2. Open customization

1. Open your new dashboard from the dashboard list.
2. Click **Customize** to start adding content.

## 3. Add widgets

Select preconfigured widgets you want to see, for example:

- **Successful calls**
- **Calls today**
- **Failed calls in the last 30 days**
- **New campaigns**

Then click **Add** for each widget.

## 4. Adjust size and view

- Resize individual widgets with the **plus** control if you want larger cards.
- Use the **expand arrow** to open a widget in full-screen view.

This lets you tailor the dashboard to your exact workflow.
