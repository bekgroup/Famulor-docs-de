Use the **integrated call-transfer tool** to forward an active call **from one assistant to another** inside Famulor. You configure it under **Prompt & Tools** in the **Integrated tools** section: pick a **target assistant** and describe **when** the handoff should happen.

## When to use it

- **Language routing** — e.g. hand off to a Spanish, Russian, or Italian specialist assistant when the model detects that language.
- **Compliance / consent** — e.g. move the caller to an assistant configured for a path where recording or handling differs from the first line.
- **Team structure** — chain specialized assistants (sales → onboarding, triage → expert) **without** provisioning a separate phone number per leg.

## Configure

1. Open the **source** assistant → **Prompt & Tools**.
2. In **Integrated tools**, enable the tool that **forwards the call to another assistant** (see product UI label).
3. Select the **target assistant**.
4. In the description field, spell out **exactly when** to transfer (clear conditions make the behavior reliable).

<Tip>
This is **assistant → assistant**, not transfer to a **human phone number**. For cold/warm transfer to a person, see [Call forwarding (cold vs warm)](/en/platform/assistant-call-forwarding).
