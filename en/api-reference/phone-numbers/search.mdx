Dieser Endpunkt ermöglicht es Ihnen, nach verfügbaren Telefonnummern von unserem Anbieter zu suchen. Verwenden Sie die Ergebnisse, um eine Nummer zu finden, die Sie kaufen möchten.

### Query-Parameter

<ParamField query="country_code" type="string" default="DE">
  Der ISO 3166-1 alpha-2 Ländercode, in dem gesucht werden soll (z.B. DE, GB, AU, CA)
</ParamField>

<ParamField query="contains" type="string">
  Filtert Telefonnummern, die bestimmte Ziffern enthalten (nur numerische Zeichen, maximal 10 Ziffern)
</ParamField>

### Antwort-Felder

<ResponseField name="data" type="array">
  Array von verfügbaren Telefonnummern
  
  <Expandable title="Verfügbare Nummer-Eigenschaften">
    <ResponseField name="phone_number" type="string">
      Die Telefonnummer im E.164-Format
    </ResponseField>

    <ResponseField name="phone_number_formatted" type="string">
      Die Telefonnummer formatiert für die Anzeige
    </ResponseField>

    <ResponseField name="country_code" type="string">
      Der ISO-Ländercode
    </ResponseField>

    <ResponseField name="price" type="number">
      Monatlicher Mietpreis in USD
    </ResponseField>

    <ResponseField name="stripe_price_id" type="string">
      Die Stripe-Preis-ID für die Abrechnung
    </ResponseField>

    <ResponseField name="address_requirements" type="string">
      Adressanforderungen für diese Nummer: `none`, `local`, `foreign` oder `inventory`
    </ResponseField>

    <ResponseField name="sms_capable" type="boolean">
      Ob die Nummer SMS unterstützt
    </ResponseField>
  </Expandable>
</ResponseField>

<CodeGroup>
```bash cURL
curl -X GET "https://app.famulor.de/api/user/phone-numbers/search?country_code=US" \
     -H "Authorization: Bearer YOUR_API_KEY" \
     -H "Content-Type: application/json"
```

```bash cURL (mit Filter)
curl -X GET "https://app.famulor.de/api/user/phone-numbers/search?country_code=DE&contains=1234" \
     -H "Authorization: Bearer YOUR_API_KEY" \
     -H "Content-Type: application/json"
```

```javascript JavaScript
const response = await fetch(
  "https://app.famulor.de/api/user/phone-numbers/search?country_code=US",
  {
    method: "GET",
    headers: {
      "Content-Type": "application/json",
      Authorization: "Bearer YOUR_API_KEY",
    },
  }
);

const data = await response.json();
```

```python Python
import requests

params = {
    "country_code": "US",
    # Optional: "contains": "1234"
}

response = requests.get(
    "https://app.famulor.de/api/user/phone-numbers/search",
    headers={
        "Content-Type": "application/json",
        "Authorization": "Bearer YOUR_API_KEY",
    },
    params=params,
    timeout=10,
)

print(response.json())
```
