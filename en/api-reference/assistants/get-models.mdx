---
title: "Get available LLM models"
api: "GET /user/assistants/models"
description: "Retrieve all available language models for assistant configuration"
---

Fetch a complete list of Large Language Models (LLMs) you can use when creating or updating your assistants.

### Response Fields

<ResponseField name="data" type="array">
  Array of available LLM models
  <Expandable title="properties">
    <ResponseField name="id" type="integer">
      Unique model identifier
    </ResponseField>

    <ResponseField name="name" type="string">
      Model display name
    </ResponseField>
  </Expandable>
</ResponseField>

### Notes

- No parameters needed for this endpoint
- Use the model `name` when creating or updating assistants
- All available LLM models are returned in a single request
- Different models offer varying capabilities, performance characteristics, and pricing
