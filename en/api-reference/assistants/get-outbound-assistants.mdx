---
title: "Get outbound assistants"
api: "GET /user/assistants/outbound"
description: "Retrieve all outbound assistants in your account"
---

Get a list of all outbound assistants you've created. Outbound assistants are designed to make proactive calls to leads and prospects.

### Response Fields

<ResponseField name="data" type="array">
  Array of outbound assistant objects
  <Expandable title="properties">
    <ResponseField name="id" type="integer">
      Unique assistant identifier
    </ResponseField>

    <ResponseField name="user_id" type="integer">
      Account owner identifier
    </ResponseField>

    <ResponseField name="name" type="string">
      Assistant name
    </ResponseField>

    <ResponseField name="type" type="string">
      Assistant type (always "outbound" for this endpoint)
    </ResponseField>

    <ResponseField name="mode" type="string">
      Engine mode (`pipeline` or `multimodal`)
    </ResponseField>

    <ResponseField name="status" type="string">
      Current assistant status
    </ResponseField>

    <ResponseField name="voice_id" type="integer">
      Identifier of the voice used
    </ResponseField>

    <ResponseField name="language_id" type="integer">
      Identifier of the language configured
    </ResponseField>

    <ResponseField name="timezone" type="string">
      Timezone configuration
    </ResponseField>

    <ResponseField name="initial_message" type="string">
      Opening message the assistant speaks
    </ResponseField>

    <ResponseField name="system_prompt" type="string">
      System prompt defining assistant behavior
    </ResponseField>

    <ResponseField name="variables" type="object">
      Custom variables defined for this assistant
    </ResponseField>

    <ResponseField name="webhook_url" type="string">
      Webhook URL for post-call notifications
    </ResponseField>

    <ResponseField name="is_webhook_active" type="boolean">
      Whether webhook notifications are enabled
    </ResponseField>

    <ResponseField name="created_at" type="string">
      Creation timestamp
    </ResponseField>

    <ResponseField name="updated_at" type="string">
      Last update timestamp
    </ResponseField>
  </Expandable>
</ResponseField>

### Notes

- This endpoint only returns assistants with type "outbound"
- All assistants belonging to your account are returned (no pagination)
- Use these assistants with the make call endpoint to start outbound calls
- Suitable for sales calls, follow-ups, surveys, and other proactive calling use cases
