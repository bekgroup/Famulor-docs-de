description: "Alle verfügbaren Sprachen für die Assistenten-Konfiguration abrufen"
---

Dieser Endpunkt gibt eine Liste aller verfügbaren Sprachen zurück, die beim Erstellen oder Aktualisieren von Assistenten verwendet werden können.

### Antwort-Felder

Die API gibt ein direktes Array von verfügbaren Sprachen zurück:

<ResponseField name="array" type="array">
  Array der verfügbaren Sprachen

  <Expandable title="Sprachen-Eigenschaften">
    <ResponseField name="id" type="integer">
      Die eindeutige Kennung der Sprache
    </ResponseField>

    <ResponseField name="name" type="string">
      Der Anzeigename der Sprache
    </ResponseField>

    <ResponseField name="iso_2" type="string">
      Der ISO 639-1 zweistellige Sprachcode
    </ResponseField>
  </Expandable>
</ResponseField>

### Hinweise

- Dieser Endpunkt benötigt keine Parameter
- Verwenden Sie das `name`-Feld beim Erstellen oder Aktualisieren von Assistenten
- Der `iso_2`-Code kann für sprachspezifische Funktionalität nützlich sein
- Alle verfügbaren Sprachen werden in einer einzigen Anfrage zurückgegeben

