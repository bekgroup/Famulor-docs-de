---
title: "Make a Call"
api: "POST https://app.famulor.de/api/user/make_call"
description: "Initiate an outbound phone call with a specific Famulor AI assistant via API. Pass lead data and trigger automated voice conversations programmatically."
---

This endpoint allows you to initiate a phone call.

### Request Body

<ParamField body="phone_number" type="string" required>
  The customer's phone number in E.164 format (e.g. +1234567890)
</ParamField>

<ParamField body="assistant_id" type="integer" required>
  The ID of the assistant to make the call with
</ParamField>

<ParamField body="variables" type="object">
  The variables to be passed to the lead

  <Expandable title="Variable Properties">
    <ParamField body="customer_name" type="string" default="<string>">
      The name of the customer
    </ParamField>

    <ParamField body="email" type="string" default="<string>">
      The customer's email address
    </ParamField>
  </Expandable>
</ParamField>

### Response

<ResponseField name="message" type="string">
  Confirmation message that the call was successfully initiated
</ResponseField>

<ResponseField name="call_id" type="integer">
  The unique ID of the created call
</ResponseField>

<ResponseField name="status" type="string">
  The initial status of the call (e.g. "initiated")
</ResponseField>


<ResponseExample>

```json 200 Success
{
  "message": "Call initiated successfully",
  "call_id": 480337,
  "status": "initiated"
}
```

```json 400 Invalid Phone Number
{
  "message": "Invalid phone number format. Please use E.164 format (e.g. +1234567890)"
}
```

```json 404 Assistant Not Found
{
  "message": "Assistant not found or does not belong to the authenticated user"
}
```

```json 422 Insufficient Balance
{
  "message": "Insufficient balance to make call. Please top up your account."
}
```

</ResponseExample>
