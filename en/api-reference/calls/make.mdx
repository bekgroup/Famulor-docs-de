This endpoint allows you to initiate a phone call.

### Request Body

<ParamField body="phone_number" type="string" required>
  The customer's phone number in E.164 format (e.g. +1234567890)
</ParamField>

<ParamField body="assistant_id" type="integer" required>
  The ID of the assistant to make the call with
</ParamField>

<ParamField body="variables" type="object">
  The variables to be passed to the lead

  <Expandable title="Variable Properties">
    <ParamField body="customer_name" type="string" default="<string>">
      The name of the customer
    </ParamField>

    <ParamField body="email" type="string" default="<string>">
      The customer's email address
    </ParamField>
  </Expandable>
</ParamField>

### Response

<ResponseField name="message" type="string">
  Confirmation message that the call was successfully initiated
</ResponseField>

<ResponseField name="call_id" type="integer">
  The unique ID of the created call
</ResponseField>

<ResponseField name="status" type="string">
  The initial status of the call (e.g. "initiated")
</ResponseField>

<CodeGroup>
```bash cURL
curl -X POST "https://app.famulor.de/api/user/make_call" \
  -H "Authorization: Bearer YOUR_API_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "phone_number": "+4915123456789",
    "assistant_id": 123,
    "variables": {
      "customer_name": "Max Mustermann",
      "email": "max.mustermann@example.com"
    }
  }'
```

```python Python
import requests

payload = {
    "phone_number": "+4915123456789",
    "assistant_id": 123,
    "variables": {
        "customer_name": "Max Mustermann",
        "email": "max.mustermann@example.com",
    },
}

response = requests.post(
    "https://app.famulor.de/api/user/make_call",
    headers={
        "Content-Type": "application/json",
        "Authorization": "Bearer YOUR_API_TOKEN",
    },
    json=payload,
    timeout=10,
)

print(response.json())
```
