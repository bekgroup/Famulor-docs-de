Willkommen zu den Grundlagen von famulor.de. Hier erkunden wir die wesentlichen Bausteine, die Sie verstehen müssen, bevor Sie Ihren KI-Assistenten und Anrufe konfigurieren.

## 1. System-Prompt & Erste Nachricht

**System-Prompt**

- Das "Gehirn" Ihres KI-Assistenten. Sie schreiben eine kurze (oder lange) Anweisung, die das Verhalten und den Tonfall des Assistenten leitet.
- Dies umfasst wichtige Geschäftsinformationen, Haftungsausschlüsse oder wie Sie möchten, dass die KI Randfälle behandelt.
- Zum Beispiel: _"Sie sind ein hilfreicher Telefonassistent für XYZ Company, Sie begrüßen freundlich, bestätigen die Informationen des Anrufers, etc."_

**Erste Nachricht**

- Das allererste, was die KI sagt.
- Typischerweise kurz, wie "Hallo, danke für Ihren Anruf bei XYZ Company!"

## 2. Stimmen

- famulor.de integriert **ElevenLabs** Stimmen für natürlich klingende Sprache.
- Sie können Ihre eigene Stimme **klonen**, indem Sie Audio-Proben bereitstellen, oder aus integrierten Stimmen wählen.
- Jede Stimme kann einstellbare Parameter wie Sprechgeschwindigkeit, Füllwortverwendung oder Akzent haben (falls verfügbar).

## 3. Alles zusammenbringen

- Ein **KI-Assistent** wird erstellt, indem ein System-Prompt, eine erste Nachricht und eine gewählte Stimme kombiniert werden.
- Optional fügen Sie Tools hinzu (wie "Anruf weiterleiten" oder "Anruf beenden").
- Sobald der Assistent eingerichtet ist, weisen Sie ihn einer Telefonnummer zu (für eingehende Anrufe) oder einer Kampagne (für ausgehende Anrufe).
