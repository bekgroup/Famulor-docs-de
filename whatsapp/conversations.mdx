WhatsApp-Chats werden zusammen mit Ihren anderen Konversationen erfasst. So können Sie den Nachrichtenverlauf einsehen, die KI-Leistung überwachen und die Kundeninteraktionen verwalten.

## WhatsApp-Chats anzeigen

WhatsApp-Chats erscheinen in Ihrem Bereich **Chats** (Conversations) mit einem „WhatsApp“-Badge.

Jedes Gespräch enthält:

* Den vollständigen Nachrichtenverlauf
* Die Telefonnummer des Kunden
* Den verbundenen KI-Assistenten
* Zeitstempel für alle Nachrichten
* Kosten- und Nutzungsmetriken

## Wie WhatsApp-Chats funktionieren

### Kunde initiiert den Kontakt

1. Der Kunde sendet eine Nachricht an Ihre WhatsApp-Nummer
2. Das System erstellt ein neues Gespräch oder findet ein bestehendes
3. Die Nachricht wird an Ihren KI-Assistenten geleitet
4. Die KI generiert eine Antwort und sendet sie ab
5. Das 24-Stunden-Messaging-Fenster öffnet sich

### Innerhalb des 24-Stunden-Fensters

* Die KI antwortet automatisch auf jede Nachricht
* Freie Antworten sind erlaubt
* Das Gespräch wird natürlich fortgesetzt

### Nach 24 Stunden

* Der Kunde muss erneut schreiben, um das Fenster wieder zu öffnen
* Sie können Template-Nachrichten senden, um den Kunden erneut anzusprechen
* Die KI bleibt bereit zu antworten, sobald der Kunde reagiert

## Chats-Funktionen

### Nachrichtenverlauf

Sehen Sie den vollständigen Austausch zwischen Kunde und KI ein:

* Eingehende Kundennachrichten
* Ausgehende KI-Antworten
* Zeitstempel für jede Nachricht
* Funktionsaufrufe, die während des Gesprächs getätigt wurden

### Gesprächsvariablen

Variablen, die während des Gesprächs gesammelt oder gesetzt wurden:

* Kunden-Identifikator
* Von der KI extrahierte Daten
* Aus Automatisierungs-Flows übergebener Kontext

### Kostenverfolgung

Jedes Gespräch erfasst:

* Gesamtzahl der Nachrichten
* Kosten pro Nachricht
* Gesamtkosten des Gesprächs

## Verwaltung von Chats

### Chatsfilter

Filtern Sie Ihre Chatsliste nach:

* Typ (WhatsApp, Web Widget, Test)
* Assistent
* Datumsbereich
* Kunden-Identifikator

### Chatsdaten exportieren

Exportieren Sie Chatsdaten zur Analyse:

1. Navigieren Sie zu „Chats“
2. Wenden Sie Filter nach Bedarf an
3. Nutzen Sie die Exportoptionen, um die Daten herunterzuladen

## Verhalten des KI-Assistenten

Ihr KI-Assistent bearbeitet Chats basierend auf seiner Konfiguration:

### System-Prompt

Derselbe System-Prompt, der für Sprachanrufe und Web-Widgets verwendet wird, gilt auch für Chats. Stellen Sie sicher, dass Ihr Prompt gut für die textbasierte Kommunikation funktioniert.

<Tip>
  **Textfreundliche Prompts** — Für Assistenten, die sowohl Sprache als auch WhatsApp bearbeiten, sollten Sie Anweisungen zur Textformatierung (z. B. Verwendung von Aufzählungspunkten und kürzeren Absätzen) in den Prompt aufnehmen.
</Tip>

### Tools und Funktionen

Die KI kann während Chats Tools aufrufen:

* Informationen in Datenbanken nachschlagen
* Termine planen
* An einen menschlichen Mitarbeiter übergeben
* Daten sammeln und validieren

### Wissensdatenbanken

Die Wissensdatenbanken Ihres Assistenten stehen für Chats zur Verfügung. So kann die KI Fragen mithilfe Ihrer hochgeladenen Inhalte beantworten.

## Best Practices

### 1. Chats regelmäßig überprüfen

Überwachen Sie die KI-Antworten, um die Qualität sicherzustellen. Achten Sie auf:

* Richtige Informationen
* Angemessenen Tonfall
* Erfolgreiche Zielerreichung
* Signale für Kundenzufriedenheit

### 2. Ihre KI trainieren

Nutzen Sie Erkenntnisse aus den Chats, um Ihren Assistenten zu verbessern:

* Identifizieren Sie häufige Fragen
* Aktualisieren Sie Wissensdatenbanken
* Verfeinern Sie System-Prompts
* Fügen Sie fehlende Tools oder Funktionen hinzu

### 3. Antwortzeiten überwachen

WhatsApp-Nutzer erwarten schnelle Antworten. Überwachen Sie:

* KI-Antwortlatenz
* Jegliche Verzögerungen oder Timeouts
* Fehlgeschlagene Nachrichtenzustellungen

### 4. Eskalationen handhaben

Konfigurieren Sie Eskalationspfade für komplexe Probleme:

* Tool zur Übergabe an menschliche Mitarbeiter
* Sammeln von Rückrufinformationen
* Planung von Follow-up-Anrufen

## Chats-Metriken

Verfolgen Sie wichtige Kennzahlen für Ihre Chats:

| Metrik                             | Beschreibung                                              |
| ---------------------------------- | --------------------------------------------------------- |
| **Gesamtzahl der Chats**           | Anzahl der eindeutigen Chats                              |
| **Nachrichten pro Chat**           | Durchschnittliche Nachrichtenanzahl                       |
| **Antwortrate**                    | Prozentsatz der Nachrichten mit KI-Antworten              |
| **Durchschnittliche Antwortzeit**  | Zeit zwischen Kundennachricht und KI-Antwort              |
| **Kosten pro Chat**                | Durchschnittliche Kosten inklusive KI-Nutzung             |

## Integration mit anderen Kanälen

WhatsApp-Chats existieren neben Ihren anderen Kommunikationskanälen:

* **Telefonanrufe**: Sprachgespräche, die separat erfasst werden
* **Web-Widget**: Textgespräche von Ihrer Website
* **API-Gespräche**: Programmgesteuerte Chat-Sitzungen

Alle Chats-Typen nutzen denselben KI-Assistenten und bieten so eine konsistente Erfahrung über alle Kanäle hinweg.
