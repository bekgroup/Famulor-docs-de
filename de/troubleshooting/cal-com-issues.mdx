> Beheben Sie häufige Cal.com-Integrationsprobleme

Lösungen für häufige Probleme mit der Cal.com-Integration.

## Verbindungsprobleme

### Ungültiger API-Key

**Problem:** Events laden nicht oder Sync schlägt fehl

**Lösung:**

- Prüfen Sie, ob der API-Key in Cal.com aktiv ist (Settings -> Developer -> API Keys)
- Key muss mit `cal_live_` beginnen
- Bei Bedarf neuen API-Key erstellen

### Events laden nicht

**Problem:** Event-Dropdown bleibt leer

**Lösung:**

- API-Key prüfen
- Außerhalb des Felds klicken, um Laden auszulösen
- Seite neu laden
- Prüfen, ob Events in Cal.com existieren

## Sync-Probleme

### Sync Event fehlgeschlagen

**Problem:** "Sync Event" schlägt fehl

**Lösung:**

1. Stattdessen **"Troubleshoot"** klicken
2. Das setzt Buchungsfelder auf korrekte Konfiguration zurück
3. Änderungen in Cal.com prüfen

### Feldkonfiguration fehlerhaft

**Problem:** Fehler zu Pflichtfeldern/fehlenden Feldern

**Lösung:**

- **"Troubleshoot"** klicken
- Pflichtfelder nur: Name, E-Mail, Telefon
- Alle anderen Felder optional

## Buchungsprobleme

### Keine Kalendereinladung gesendet

**Problem:** Buchung erfolgreich, Kunde erhält keine Einladung

**Lösung:**

- E-Mail-Variable in [Assistant Prompt & Tools](/de/ai-assistants/prompt-and-tools) definieren
- E-Mail beim Lead-Import oder Lead-Erstellen mitgeben
- E-Mail-Format prüfen

### Telefonnummer fehlt

**Problem:** Telefonnummer erscheint nicht im Kalenderevent

**Lösung:**

- **"Sync Event"** oder **"Troubleshoot"** klicken
- Telefonfeld wird automatisch ergänzt
- Danach wird Nummer korrekt übertragen

### Zeitzonenfehler

**Problem:** Termine zur falschen Zeit geplant

**Lösung:**

- Zeitzone in Cal.com Event-Einstellungen prüfen
- Sicherstellen, dass der Assistent die richtige Zeit versteht
- Mit konkreten Zeitangaben testen

## Testprobleme

### Web-Call Buchungen schlagen fehl

**Lösung:**

- Mit echten Telefonanrufen testen (nicht Web-Call)
- Web-Calls übergeben Telefonnummer teils unzuverlässig
- Bei Web-Call Nummer im internationalen Format mit `+` angeben

### Mehrere Kalender funktionieren nicht

**Lösung:**

- Je Kalender ein eigenes Event auswählen
- Eindeutige Event-Namen verwenden (z. B. `15min`, `30min`)
- Jedes Event einzeln synchronisieren

## Hilfe

Wenn das Problem bleibt:

1. Call-Transkripte auf Buchungsversuche prüfen
2. Event im Cal.com Dashboard verifizieren
3. Mit einfachem Telefonanruf testen
4. Support kontaktieren mit:
- Fehler-Screenshot
- Screenshot der Cal.com Event-Einstellungen
- Transkript mit Buchungsversuch
