---
title: "Mid-call Action löschen"
api: "DELETE https://app.famulor.de/api/user/tools/{id}"
description: "Entfernen Sie eine Mid-Call Action dauerhaft aus Ihrem Famulor-Konto per API. Trennt das Tool von allen KI-Telefonassistenten, die es aktuell verwenden."
---


Dieser Endpunkt ermöglicht es Ihnen, ein Mid-call Action dauerhaft zu löschen. Dadurch wird das Tool auch von allen Assistenten entfernt, denen es zugewiesen war.

### Pfad-Parameter

<ParamField path="id" type="integer" required>
  Die eindeutige Kennung des zu löschenden Tools
</ParamField>

### Antwortfelder

<ResponseField name="message" type="string">
  Erfolgsmeldung, die die Löschung bestätigt
</ResponseField>

<ResponseExample>

```json 200 OK
{
  "message": "Tool deleted successfully"
}
```

```json 404 Not Found
{
  "message": "Tool not found"
}
```

</ResponseExample>

### Hinweis

Wenn ein Tool gelöscht wird, wird es automatisch von allen Assistenten entfernt. Um Tool-Zuweisungen zu verwalten, ohne das Tool zu löschen, verwenden Sie:

- **[Assistent aktualisieren](/de/api-reference/assistants/update)** - Verwenden Sie den `tool_ids`-Parameter, um Tools von bestimmten Assistenten zu entfernen
