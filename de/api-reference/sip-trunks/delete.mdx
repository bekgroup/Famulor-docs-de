---
title: "SIP-Trunk löschen"
api: "DELETE https://app.famulor.de/api/user/phone-numbers/{id}"
description: "SIP-Trunk löschen und Voice-Infrastruktur bereinigen"
---

Löscht einen SIP-Trunk und entfernt die zugehörigen Inbound- und Outbound-Trunks aus der Voice-Infrastruktur.

Hier wird derselbe Endpunkt wie beim Freigeben allgemeiner Telefonnummern verwendet. Bei SIP-Trunks erfolgt die Löschung sofort (kein Abrechnungszeitraum).

<Note>
  Siehe [Telefonnummer freigeben](/de/api-reference/phone-numbers/release) für die vollständige Endpunkt-Dokumentation. SIP-Trunks werden sofort gelöscht und liefern `"Phone number deleted successfully."` zurück.
</Note>

### Pfad-Parameter

<ParamField path="id" type="integer" required>
  Die eindeutige Kennung des zu löschenden SIP-Trunks
</ParamField>

### Antwort

<ResponseExample>
  ```json 200 Deleted theme={null}
  {
    "message": "Phone number deleted successfully."
  }
  ```

  ```json 404 Not Found theme={null}
  {
    "error": "Phone number not found."
  }
  ```
</ResponseExample>

### Was beim Löschen passiert

Wenn du einen SIP-Trunk löschst:

* **Inbound- und Outbound-Trunks** werden aus der Voice-Infrastruktur entfernt
* **Assistenten**, die diesen SIP-Trunk nutzen, verlieren die zugewiesene Telefonnummer
* **Kampagnen**, die diese Nummer nutzen, entfernen sie aus der Rotation
* **Anrufhistorie** bleibt erhalten und ist weiterhin abrufbar
