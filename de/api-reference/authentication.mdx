Die Famulor API verwendet API-Schlüssel zur Authentifizierung von Anfragen. Du musst deinen API-Schlüssel im Header jeder Anfrage an die API einbeziehen.

<Info>
Für eine umfassende Anleitung mit Code-Beispielen, Sicherheits-Best-Practices und Fehlerbehebung siehe unseren [vollständigen Authentifizierungs-Leitfaden](/de/developers/authentication-guide).
</Info>

## API-Schlüssel erhalten

Um einen API-Schlüssel zu erhalten:

1. Melde dich bei deinem Famulor-Konto an unter [https://app.famulor.de](https://app.famulor.de)
2. Navigiere zur "API-Schlüssel" Seite
3. Klicke auf den "Neuen API-Schlüssel erstellen" Button
4. Kopiere deinen neuen API-Schlüssel und speichere ihn sicher

<Warning>
  Halte deinen API-Schlüssel vertraulich und sicher. Teile ihn nicht öffentlich und committe ihn nicht in Versionskontrollsysteme.
</Warning>

## API-Schlüssel verwenden

Füge deinen API-Schlüssel in den `Authorization` Header jeder Anfrage ein:

```
Authorization: Bearer YOUR_API_KEY
```

Ersetze `YOUR_API_KEY` durch deinen tatsächlichen API-Schlüssel.

## Beispiel-Anfrage

Hier ist ein Beispiel, wie du den API-Schlüssel in einer cURL-Anfrage einbeziehst:

```bash
curl -X GET "https://app.famulor.de/api/user/campaigns" \
     -H "Authorization: Bearer YOUR_API_KEY"
```

Ersetze `YOUR_API_KEY` durch deinen tatsächlichen API-Schlüssel bei Anfragen.
