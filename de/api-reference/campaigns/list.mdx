---
title: "Kampagnen auflisten"
api: "GET https://app.famulor.de/api/user/campaigns"
description: "Listen Sie alle Outbound-Anrufkampagnen Ihres Famulor-Kontos per API auf. Status, Lead-Zahlen, zugewiesene KI-Telefonassistenten und Fortschritt im Überblick."
---


Dieser Endpunkt ermöglicht es Ihnen, alle Kampagnen aufzulisten.

### Antwort-Felder

Die API gibt ein direktes Array von Kampagnen zurück:

<ResponseField name="array" type="array">
  Array aller Kampagnen

  <Expandable title="Kampagnen-Eigenschaften">
    <ResponseField name="id" type="integer">
      Die ID der Kampagne
    </ResponseField>

    <ResponseField name="name" type="string">
      Der Name der Kampagne
    </ResponseField>

    <ResponseField name="status" type="string">
      Der Status der Kampagne
    </ResponseField>

    <ResponseField name="max_calls_in_parallel" type="integer">
      Maximale Anzahl von Anrufen, die parallel getätigt werden können
    </ResponseField>

    <ResponseField name="mark_complete_when_no_leads" type="boolean">
      Ob die Kampagne als abgeschlossen markiert werden soll, wenn keine Leads vorhanden sind
    </ResponseField>

    <ResponseField name="allowed_hours_start_time" type="string">
      Die Startzeit für erlaubte Anrufzeiten
    </ResponseField>

    <ResponseField name="allowed_hours_end_time" type="string">
      Die Endzeit für erlaubte Anrufzeiten
    </ResponseField>

    <ResponseField name="allowed_days" type="array">
      Die Wochentage, an denen Anrufe erlaubt sind
    </ResponseField>

    <ResponseField name="max_retries" type="integer">
      Maximale Anzahl von Wiederholungsversuchen für fehlgeschlagene Anrufe
    </ResponseField>

    <ResponseField name="retry_interval" type="integer">
      Intervall in Sekunden zwischen Wiederholungsversuchen
    </ResponseField>

    <ResponseField name="created_at" type="string">
      Der Zeitstempel, wann die Kampagne erstellt wurde
    </ResponseField>

    <ResponseField name="updated_at" type="string">
      Der Zeitstempel, wann die Kampagne zuletzt aktualisiert wurde
    </ResponseField>
  </Expandable>
</ResponseField>
