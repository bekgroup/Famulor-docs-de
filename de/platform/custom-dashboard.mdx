## 1. Neues Custom Dashboard erstellen

1. Klicke in deinem Account auf **Custom Dashboard erstellen**.
2. Vergib einen Namen für das Dashboard.
3. Wähle eine Darstellungsart (z. B. Kacheln oder Tabelle).
4. Erstelle das Dashboard.

## 2. Anpassung öffnen

1. Öffne dein neues Dashboard in der Dashboard-Liste.
2. Klicke auf **Anpassen**, um Inhalte hinzuzufügen.

## 3. Widgets hinzufügen

Wähle die vorgefertigten Widgets aus, die du sehen möchtest, zum Beispiel:

- **Erfolgreiche Anrufe**
- **Anrufe heute**
- **Fehlgeschlagene Anrufe in den letzten 30 Tagen**
- **Neue Kampagnen**

Klicke danach jeweils auf **Hinzufügen**.

## 4. Größe und Ansicht anpassen

- Vergrößere einzelne Widgets über die **Plus**-Funktion.
- Öffne ein Widget über den **Pfeil** in der Vollbildansicht.

So passt du dein Dashboard genau an deinen Arbeitsablauf an.
