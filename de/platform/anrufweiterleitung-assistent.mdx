## Wofür das gedacht ist

- **Sprach-Routing** — z. B. Weiterleitung an einen spanischen, russischen oder italienischen Spezial-Assistenten, sobald die Sprache passt.
- **DSGVO / Einwilligung** — z. B. Wechsel auf einen Assistenten mit anderem Aufzeichnungs- oder Bearbeitungspfad.
- **Team-Struktur** — Spezialisten-Ketten (Qualifizierung → Experte) **ohne** pro Stufe eine eigene Rufnummer.

## Einrichtung

1. **Quell-Assistent** öffnen → **Prompt & Werkzeuge**.
2. Bei **integrierten Tools** das Tool aktivieren, mit dem der Anruf **an einen anderen Assistenten** geht (Bezeichnung in der Oberfläche).
3. **Ziel-Assistenten** auswählen.
4. Im Beschreibungsfeld **konkret festlegen**, unter welchen Bedingungen weitergeleitet wird — je präziser, desto stabiler.

<Tip>
Das ist **Assistent → Assistent**, keine Weiterleitung an eine **menschliche Rufnummer**. Für Kalt-/Warmanruf an eine Person siehe [Anrufsweiterleitung (Kalt vs. Warm)](/de/platform/anrufsweiterleitung).
