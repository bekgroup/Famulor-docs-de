# MCP Client einrichten

Verbinde den gehosteten Famulor-MCP-Server mit deinem KI-Client in wenigen Schritten. Nach der Einrichtung meldest du dich im Browser mit deinem [Famulor API-Schlüssel](https://app.famulor.de/api-keys) an — der Schlüssel gehört **nicht** in Konfigurationsdateien.

## Server-URL

**Kanonischer Endpunkt:** `https://mcp.famulor.io/mcp`

Sowohl `https://mcp.famulor.io/` als auch `https://mcp.famulor.io/mcp` funktionieren — füge eine der URLs im Connector-Dialog deines Clients ein.

## Schnellstart

1. Server im Client hinzufügen (siehe [ChatGPT](#chatgpt), [Claude](#claude) oder [weitere Clients](#weitere-clients)).
2. Bei Aufforderung OAuth-Anmeldung im Browser abschließen und deinen Famulor API-Schlüssel einfügen.
3. In natürlicher Sprache fragen, z. B. *„Zeige meine letzten 10 Anrufe“* oder *„Liste meine Assistenten“*.

<Note>
Die Authentifizierung nutzt MCP OAuth 2.1 (PKCE). Dein API-Schlüssel wird **nicht** in einer Datenbank gespeichert — nur verschlüsselt im Access Token auf deinem Gerät. Tokens laufen nach 30 Tagen ab; der Client fordert dich zur erneuten Anmeldung auf.
</Note>

## ChatGPT

Famulor ist in **ChatGPT** (Web) als **eigene App** verfügbar. Du benötigst einen ChatGPT-Plan mit Apps/Connectors und aktivierten **Entwicklermodus**.

### Famulor-App hinzufügen

1. **ChatGPT** → **Einstellungen** → **Apps** (oder **Connectors**).
2. **App erstellen** / **Neue App** wählen.
3. Ausfüllen:

| Feld | Wert |
| --- | --- |
| **Name** | `Famulor` |
| **Beschreibung** | `Famulor Voice Agent` (optional) |
| **URL des MCP-Servers** | `https://mcp.famulor.io` oder `https://mcp.famulor.io/mcp` |
| **Authentifizierung** | **OAuth** |

4. Sicherheitshinweis bestätigen und **Erstellen** klicken.
5. OAuth-Anmeldung abschließen und deinen [API-Schlüssel](https://app.famulor.de/api-keys) einfügen.

<img
  src="/images/mcp/chatgpt-add-app.png"
  alt="Famulor als eigene MCP-App in ChatGPT mit OAuth hinzufügen"
/>

### Famulor im Chat nutzen

1. Neuen Chat starten und bei Bedarf **Entwicklermodus** aktivieren.
2. **Famulor** im Tools-/Apps-Menü des Eingabefelds auswählen.
3. Fragen zu deinem Konto stellen — z. B. *„Was waren meine letzten Anrufe?“*

  src="/images/mcp/chatgpt-famulor-calls.png"
  alt="ChatGPT listet mit Famulor MCP die letzten Anrufe auf"
/>

## Claude

Famulor funktioniert in **Claude Desktop** (eigener Connector) und **Claude Code** (CLI). Beide nutzen OAuth — melde dich mit deinem [API-Schlüssel](https://app.famulor.de/api-keys) an, wenn du dazu aufgefordert wirst.

### Claude Desktop

1. **Claude** → **Einstellungen** → **Connectors** öffnen.
2. **Benutzerdefinierten Connector hinzufügen** wählen.
3. Ausfüllen:

| Feld | Wert |
| --- | --- |
| **Name** | `Famulor` |
| **URL** | `https://mcp.famulor.io/mcp` |

4. **Hinzufügen** klicken und OAuth-Anmeldung im Browser abschließen.

<img
  src="/images/mcp/claude-add-connector.png"
  alt="Famulor als benutzerdefinierten MCP-Connector in Claude Desktop hinzufügen"
/>

### Famulor in Claude nutzen

1. Neuen Chat starten und den Connector **Famulor** für die Konversation aktivieren.
2. In natürlicher Sprache fragen — z. B. *„Nutze Famulor und zeige mir meine letzten Anrufe“*.

<img
  src="/images/mcp/claude-famulor-calls.png"
  alt="Claude listet mit Famulor MCP die letzten Anrufe in einer Tabelle auf"
/>

### Claude Code

```bash
claude mcp add --transport http famulor https://mcp.famulor.io/mcp
```

Server zuerst hinzufügen — bei Aufforderung mit deinem [API-Schlüssel](https://app.famulor.de/api-keys) anmelden.

## Weitere Clients

**Global:** Cursor **Einstellungen → Tools & MCP → New MCP Server** (öffnet `~/.cursor/mcp.json`)

**Projekt:** `.cursor/mcp.json` im Projektroot anlegen.

```json
{
  "mcpServers": {
    "famulor": {
      "type": "http",
      "url": "https://mcp.famulor.io/mcp"
    }
  }
}
```

Nach dem Speichern MCP in den Einstellungen aktualisieren. Bei Aufforderung mit deinem [API-Schlüssel](https://app.famulor.de/api-keys) anmelden.

### Windsurf

`~/.codeium/windsurf/mcp_config.json`

```json
{
  "mcpServers": {
    "famulor": {
      "serverUrl": "https://mcp.famulor.io/mcp"
    }
  }
}
```

### VS Code

`.vscode/mcp.json`

```json
{
  "servers": {
    "famulor": {
      "type": "http",
      "url": "https://mcp.famulor.io/mcp"
    }
  }
}
```

### Cline

**Cline-Erweiterung → MCP Servers → Configure**

```json
{
  "mcpServers": {
    "famulor": {
      "url": "https://mcp.famulor.io/mcp"
    }
  }
}
```

### Continue

`~/.continue/config.yaml`

```yaml
mcpServers:
  - name: famulor
    type: streamable-http
    url: "https://mcp.famulor.io/mcp"
```

### Zed

`~/.config/zed/settings.json`

```json
{
  "context_servers": {
    "famulor": {
      "command": "npx",
      "args": ["-y", "mcp-remote@latest", "https://mcp.famulor.io/mcp"]
    }
  }
}
```

## Fehlerbehebung

### OAuth / Anmeldung schlägt fehl

- [API-Schlüssel](https://app.famulor.de/api-keys) prüfen (ohne Leerzeichen).
- Erneut anmelden — Tokens laufen nach 30 Tagen ab.
- Bei ChatGPT: **Entwicklermodus** aktivieren und **OAuth** wählen (kein API-Key in Headern).

### Server verbindet nicht

- `https://mcp.famulor.io/mcp` (oder `https://mcp.famulor.io/`) verwenden.
- Health-Check: [https://mcp.famulor.io/health](https://mcp.famulor.io/health)
- Client neu starten; bei Cursor: **Einstellungen → MCP → Aktualisieren**.

### Tools erscheinen nicht

- OAuth-Anmeldung nach dem Hinzufügen des Servers abschließen.
- In ChatGPT: App **Famulor** für den aktuellen Chat auswählen.
