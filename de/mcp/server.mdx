## Übersicht

Der Famulor-MCP-Server stellt die gesamte Famulor-Plattform als **76 Tools** bereit, die dein KI-Assistent aufrufen kann. Frag in natürlicher Sprache — das Modell wählt die passenden Tools. Alle Tools liefern strukturierte Ausgaben (Output-Schemas + `structuredContent`) und Title-Annotations für Directory-kompatible Clients.

**Beispiel-Prompts:**

- *„Erstelle einen deutschen Vertriebsassistenten mit GPT-4.1-mini und der Stimme Susi“*
- *„Starte Kampagne #42 und füge diese 50 Leads hinzu“*
- *„Zeige alle Anrufe von heute, bei denen das Ziel nicht erreicht wurde“*
- *„Lege einen Ordner ‚Kunden‘ an und verschiebe alle Inbound-Assistenten hinein“*

Mehr Inspiration: [Use Cases & Prompts](/de/mcp/use-cases) — gruppierte Anwendungsfälle mit fertigen Prompts und Best Practices.

<Note>
Neu bei MCP? Starte mit [MCP Client einrichten](/de/mcp/client) — Server-URL hinzufügen, dann per OAuth und API-Schlüssel im Browser anmelden.
</Note>

## Gehostete Endpunkte

| Endpunkt | URL |
| --- | --- |
| **MCP (kanonisch)** | `https://mcp.famulor.io/mcp` |
| **Basis-URL** | `https://mcp.famulor.io/` (ebenfalls gültig) |
| **Health** | [https://mcp.famulor.io/health](https://mcp.famulor.io/health) |
| **OAuth-Metadaten** | `https://mcp.famulor.io/.well-known/oauth-authorization-server` |

Sowohl `https://mcp.famulor.io/` als auch `https://mcp.famulor.io/mcp` funktionieren in Connector-Dialogen.

## Was du steuern kannst

| Bereich | Funktionen |
| --- | --- |
| **Assistenten** | Voice Agents anlegen, aktualisieren, löschen · Stimmen, Sprachen, LLM-/Multimodal-Modelle, TTS/STT · Webhooks für eingehende Anrufe und Gesprächsende · per `folder_id` / `label_ids` organisieren |
| **Ordner & Labels** | Assistenten in Ordner organisieren (ein Ordner pro Assistent) · Labels mit Farben taggen (mehrere pro Assistent) · anlegen, umbenennen, löschen |
| **Anrufe** | Outbound-Anrufe · Anrufliste abrufen, Details, löschen |
| **Kampagnen** | Kampagnen anlegen, Retry-Regeln, erlaubte Zeitfenster · starten, stoppen, löschen |
| **Leads** | Leads hinzufügen (inkl. Sekundärkontakte) · Status und Variablen · löschen |
| **Wissensdatenbanken** | Anlegen · Website-Dokumente · aktualisieren / löschen |
| **Telefonnummern** | Suche nach Land und Vorwahl · kaufen · Nickname setzen (`update_phone_number`) · freigeben · SIP-Trunk CRUD |
| **Konversationen** | Chats listen, lesen, beantworten · KI pro Konversation ein/aus (Human Takeover) |
| **WhatsApp** | Sender und Templates · 24h-Fenster · Template- oder Freitext-Nachrichten |
| **SMS** | Versand über gekaufte Nummern |
| **Mid-Call-Tools** | Eigene HTTP-Integrationen während des Gesprächs |
| **AI Replies** | Kontextbezogene Antworten für externe Chat-Plattformen |

## Authentifizierung

Der Server nutzt **MCP OAuth 2.1** (RFC 8414 Metadaten, RFC 7591 Dynamic Client Registration, PKCE S256):

1. Dein MCP-Client öffnet einen Browser-Tab.
2. Du fügst deinen [Famulor API-Schlüssel](https://app.famulor.de/api-keys) ein.
3. Der Client erhält ein Access Token; der Server verschlüsselt es mit **AES-256-GCM**.

**Dein API-Schlüssel landet in keiner Datenbank** — nur im Token, den dein Client speichert. Tokens laufen nach **30 Tagen** ab; der Client fordert zur erneuten Anmeldung auf.

<Warning>
API-Schlüssel nicht in `mcp.json`, geteilte Umgebungsvariablen oder Versionskontrolle legen. Anmeldung immer über OAuth im Client.
</Warning>

## Self-Hosting

Der gehostete Server unter `https://mcp.famulor.io` ist sofort nutzbar. Eigene Instanz:

### Deployment auf Vercel

```bash
git clone https://github.com/bekservice/Famulor-MCP.git
cd Famulor-MCP
vercel --prod
```

Umgebungsvariablen im Vercel-Projekt:

| Variable | Pflicht | Wert |
| --- | --- | --- |
| `MCP_SECRET` | ja | `openssl rand -hex 32` — verschlüsselt OAuth-Tokens |
| `MCP_ISSUER` | empfohlen | Öffentliche URL, z. B. `https://mcp.example.com` |

Domain auf das Deployment zeigen. Details in [DEPLOYMENT.md](https://github.com/bekservice/Famulor-MCP/blob/main/DEPLOYMENT.md) im Repository (inkl. stdio-Modus).

### Lokale Entwicklung

```bash
npm install
echo "MCP_SECRET=$(openssl rand -hex 32)" > .env
npm run dev:http
# Server unter http://localhost:8787
```

```bash
curl http://localhost:8787/health
curl http://localhost:8787/.well-known/oauth-authorization-server
```

## Über MCP

Das [Model Context Protocol (MCP)](https://modelcontextprotocol.io/) verbindet KI-Assistenten sicher mit externen Tools und Daten. Der Famulor-MCP-Server mappt Plattform-APIs auf MCP-Tools — damit ChatGPT, Claude und andere Clients deine Voice Agents verwalten können, nach OAuth-Anmeldung mit deinem API-Schlüssel.
