# 3CX

> So verbindest du deine 3CX-Anlage mit Famulor

<Info>
  **Wichtig:** 3CX-Anlagen haben viele Einstellungen auf Hosting- und Carrier-Ebene. Diese können die Verbindung beeinflussen.

  Wir unterstützen dich gern über [support@famulor.io](mailto:support@famulor.io), können aber wegen der eingeschränkten Einstellmöglichkeiten auf unserer Seite keine vollständige Funktionsgarantie geben.
</Info>

### Erstelle einen Trunk in 3CX

Navigiere in der 3CX-Admin-Oberfläche zu **"Trunks & Chat"** und erstelle über **+Trunk** einen neuen Trunk.

<Frame>
  <img src="/images/famulor_3cx_trunk_create_new.png" alt="3CX Trunk erstellen" />
</Frame>

Wähle die Option **"Generic SIP Trunk (IP Based)"**. Achtung: Diese Option ist nur bei gehosteten 3CX-Anlagen verfügbar.

<Frame>
  <img src="/images/famulor_3cx_trunk_creation.png" alt="Generic SIP Trunk auswählen" />
</Frame>

### Konfiguriere den 3CX-Trunk

1. Wähle einen beliebigen Namen.
2. Füge die Nummer ein, über die Famulor intern erreichbar sein soll (später mit Prefix in Regeln).
3. Trage bei **Registrar** deine individuelle Famulor-SIP-Adresse ein.

<Note>
  **So findest du deine SIP-Adresse in Famulor:**  
  `Telefonnummern` → `SIP-Trunk integrieren` → `Unsere SIP-Adresse`
</Note>

<Frame>
  <img src="/images/famulor_3cx_trunk_config.png" alt="3CX Trunk konfigurieren" />
</Frame>

### Schränke das Transportprotokoll auf TCP ein

<Frame>
  <img src="/images/famulor_3cx_tcp_protocol.png" alt="TCP als Transportprotokoll" />

### Optional: Korrekte Anrufernummer an Famulor übergeben

Wenn Famulor hinter einer 3CX-Telefonanlage verwendet wird, kann statt der echten Anrufernummer die Nummer der 3CX-Anlage übermittelt werden.  
Um das zu vermeiden, setze im **SIP-Trunk** die Felder **"From: Display-Name"** und **"Remote-Party-ID"** auf `OriginatorCallerID`.

<Frame>
  <img src="/images/famulor_3cx_caller_id_settings.png" alt="OriginatorCallerID setzen" />
</Frame>

### Erstelle die SIP-Nummer in Famulor

Nachdem der Trunk mit deiner 3CX-Konfiguration eingerichtet ist, musst du die Nummer in Famulor importieren, damit wir wissen, wie der Anruf weitergeleitet werden soll.

1. Öffne dein Famulor.ai Dashboard
2. Navigiere zu **Telefonnummern**
3. Klicke auf "SIP Trunk integrieren"

   <Frame>
     <img
       src="/resources/twillio-import-famulor-sip1.png"
       alt="Nummern in Famulor importieren"
     />
   </Frame>

4. Trage die Daten entsprechend deiner 3CX-Konfiguration ein:
   ```
   Nummer / Name: Verwende exakt dieselbe Nummer wie in der 3CX-Konfiguration (z. B. +49 151 12345678).
   Termination URL: Die Adresse deiner 3CX-Anlage, falls ausgehende Anrufe getätigt werden sollen.
   Username: Leer lassen, wenn kein Username konfiguriert wurde.
   Passwort: Leer lassen, wenn kein Passwort konfiguriert wurde.
   ```

   <Frame>
     <img src="/resources/twillio-outbound-settings.png" alt="Ausgehende Einstellungen konfigurieren" />
   </Frame>

Falls die Nummer bereits vergeben ist, wähle eine andere.

### Erstelle eine ausgehende Regel in 3CX

Erstelle jetzt eine ausgehende Regel, die definiert, wann dieser Trunk verwendet wird.  
Du kannst z. B. `999` als Präfix nutzen; wichtig ist, dass die Ziffern danach wieder entfernt werden.

<Frame>
  <img src="/images/famulor_3cx_outbound_rule.png" alt="Ausgehende Regel erstellen" />
</Frame>

**Wichtig:** Die ausgehende Regel muss oberhalb anderer Regeln stehen, die sonst greifen würden.

<Frame>
  <img src="/images/famulor_3cx_outbound_rule_order.png" alt="Reihenfolge der ausgehenden Regeln" />
</Frame>
