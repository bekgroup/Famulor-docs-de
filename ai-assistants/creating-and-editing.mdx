Bereit, Ihren ersten KI-Assistenten einzurichten? Folgen Sie diesen Schritten:

## 1. Gehen Sie zu "Assistenten"

In Ihrem famulor.de Dashboard finden Sie die **Assistenten**-Seite (manchmal als "KI-Assistenten" oder ähnlich bezeichnet).

## 2. Klicken Sie auf "Neuen Assistenten erstellen"

- Wählen Sie **Eingehend**, wenn er eingehende Anrufe beantworten soll.
- Wählen Sie **Ausgehend**, wenn er für Kampagnen ist, die Leads/Kunden anrufen.

## 3. Füllen Sie grundlegende Informationen aus

- **Assistenten-Name**: Internes Label, z.B. "Support Bot" oder "Sales Bot."
- **[System-Prompt](/ai-assistants/system-prompt)**: Skizzieren Sie die Rolle, den Tonfall und die wichtigsten Anweisungen des Assistenten.
- **[Erste Nachricht](/ai-assistants/initial-message)**: Die erste Zeile, die die KI sprechen wird (eingehende Anrufe benötigen normalerweise eine schnelle Begrüßung).

## 4. Sprache & Stimme

- Wählen Sie eine **[Stimme](/ai-assistants/voice-selection)** aus der integrierten Bibliothek oder wählen Sie eine geklonte Stimme, wenn Sie eine erstellt haben.
- (Optional) Konfigurieren Sie Sprechgeschwindigkeit, Füllwortverwendung oder "Voice Activity Detection", wenn Sie spezialisierte Einstellungen benötigen.

## 5. Tools & Funktionen

- Fügen Sie Elemente hinzu oder entfernen Sie sie wie **Anruf weiterleiten**, **Anruf beenden** oder **Terminplanung** (mehr in [Tools & Funktionen](/ai-assistants/tools-and-functions)).

## 6. Speichern & Testen

- Klicken Sie auf **Speichern** und Ihr Assistent ist bereit!
- Für eingehende Anrufe müssen Sie eine [Telefonnummer zuweisen](/phone-numbers/assigning-to-assistants).
- Für ausgehende Anrufe fügen Sie den Assistenten einer [Kampagne](/outbound-calls/campaigns) hinzu oder [testen](/ai-assistants/testing) Sie mit Ihrer eigenen Telefonnummer.

## Assistenten bearbeiten

- Kehren Sie zu **Assistenten** im Dashboard zurück.
- Klicken Sie auf den Assistenten-Namen.
- Passen Sie Prompts, Stimme an oder fügen Sie neue Tools hinzu—dann **Speichern**.
