# Web Widget

<Note>
  Das Famulor einbettbare Chat Widget ist ein produktionsreifes, anpassbares und sicheres Chat Widget für Websites. Das Widget kann über einen einzigen `<script>` Tag eingebettet werden—kein Backend-Proxy erforderlich.
</Note>

## Übersicht

<Card
  title="Was ist das Famulor Web Widget?"
  icon="widget"
  color="blue">
  <p>Das Famulor Web Widget ermöglicht es Ihren Website-Besuchern, direkt über einen Voice-Chat-Button mit Ihrem KI-Assistenten zu interagieren. Das Widget ist vollständig anpassbar und kann nahtlos in jede Website integriert werden.</p>
</Card>

## Script-Tag Integration

<Card
  title="Einfache Integration"
  icon="code"
  color="green">
  <p>Kopieren Sie den generierten Script-Tag und fügen Sie ihn in Ihre Website ein:</p>
</Card>

### Beispiel Script-Tag

<CodeGroup>
  <CodeGroupItem title="HTML Script Tag">
    ```html
    <script 
      src="https://app.famulor.de/embed.js" 
      data-assistant-id="5c92d914-9099-4d5-8f09-096ada2ade0" 
      data-position="right-bottom" 
      data-color="#3B82F6" 
      data-header-title="Voice AI" 
      data-header-subtitle="Ready to help ✨" 
      data-button-main-text="Need help?" 
      data-button-sub-text="Start voice chat ✨" 
      data-start-button-text="Start Voice Chat" 
      data-modal-title="Ready to chat?" 
      data-modal-description="Click below to start your voice conversation">
    </script>
    ```
  </CodeGroupItem>
</CodeGroup>

<Warning>
  **Verwendung:** Fügen Sie diesen Script-Tag überall in den HTML-Code Ihrer Website ein, vorzugsweise vor dem schließenden `</body>`-Tag.
</Warning>

## Data-Attribute Erklärung

<Card
  title="Verfügbare Attribute"
  icon="list"
  color="blue">
  <ul>
    <li><strong>data-assistant-id:</strong> Eindeutige ID Ihres Assistenten</li>
    <li><strong>data-position:</strong> Position des Widgets (z.B. "right-bottom")</li>
    <li><strong>data-color:</strong> Primärfarbe des Widgets</li>
    <li><strong>data-header-title:</strong> Titel im Widget-Header</li>
    <li><strong>data-header-subtitle:</strong> Untertitel im Widget-Header</li>
    <li><strong>data-button-main-text:</strong> Haupt-Button-Text</li>
    <li><strong>data-button-sub-text:</strong> Unter-Button-Text</li>
    <li><strong>data-start-button-text:</strong> Text des Start-Buttons</li>
    <li><strong>data-modal-title:</strong> Modal-Titel</li>
    <li><strong>data-modal-description:</strong> Modal-Beschreibung</li>
  </ul>
</Card>

## Häufige Probleme und Lösungen

<Card
  title="Troubleshooting"
  icon="help-circle"
