# Web Widget

<Note>
  Das Famulor einbettbare Chat Widget ist ein produktionsreifes, anpassbares und sicheres Chat Widget für Websites. Das Widget kann über einen einzigen `<script>` Tag eingebettet werden—kein Backend-Proxy erforderlich.
</Note>

## Übersicht

<Card title="Was ist das Famulor Web Widget?" color="blue">
  <p>
  Das Famulor Web Widget ermöglicht es Ihren Website-Besuchern, direkt über einen Voice-Chat-Button mit Ihrem KI-Assistenten zu interagieren. Das Widget ist vollständig anpassbar und kann nahtlos in jede Website integriert werden.</p>
</Card>

## Script-Tag Integration

<Card title="Einfache Integration" icon="code" color="green">
  <p>
  Kopieren Sie den generierten Script-Tag und fügen Sie ihn in Ihre Website ein:</p>
</Card>

### Beispiel Script-Tag

<CodeGroup>

```
<script 
  src="https://app.famulor.de/embed.js" 
  data-assistant-id="5c92d914-9099-4d5-8f09-096ada2ade0" 
  data-position="right-bottom" 
  data-color="#3B82F6" 
  data-header-title="Voice AI" 
  data-header-subtitle="Ready to help ✨" 
  data-button-main-text="Need help?" 
  data-button-sub-text="Start voice chat ✨" 
  data-start-button-text="Start Voice Chat" 
  data-modal-title="Ready to chat?" 
  data-modal-description="Click below to start your voice conversation">
</script>
```

</CodeGroup>

<Warning>
  **Verwendung:** Fügen Sie diesen Script-Tag überall in den HTML-Code Ihrer Website ein, vorzugsweise vor dem schließenden `</body>`-Tag.
</Warning>

## Data-Attribute Erklärung

<Card title="Verfügbare Attribute" icon="list" color="blue">
  <ul>
  <li>
  data-assistant-id:

   Eindeutige ID Ihres Assistenten</li>

  
  <li>
  data-position:

   Position des Widgets (z.B. "right-bottom")</li>

  
  <li>
  data-color:

   Primärfarbe des Widgets</li>

  
  <li>
  data-header-title:

   Titel im Widget-Header</li>

  
  <li>
  data-header-subtitle:

   Untertitel im Widget-Header</li>

  
  <li>
  data-button-main-text:

   Haupt-Button-Text</li>

  
  <li>
  data-button-sub-text:

   Unter-Button-Text</li>

  
  <li>
  data-start-button-text:

   Text des Start-Buttons</li>

  
  <li>
  data-modal-title:

   Modal-Titel</li>

  
  <li>
  data-modal-description:

   Modal-Beschreibung</li>

  </ul>
</Card>

## Häufige Probleme und Lösungen

<Card title="Troubleshooting" icon="help-circle" color="red">
  <ul>
  <li>
  Widget wird nicht angezeigt:

   Überprüfen Sie, ob der Script-Tag korrekt eingebettet wurde</li>

  
  <li>
  Farben falsch:

   Stellen Sie sicher, dass der Hex-Code (#) korrekt formatiert ist</li>

  
  <li>
  Position passt nicht:

   Verwenden Sie die verfügbaren Positionswerte</li>

  
  <li>
  Text wird nicht angezeigt:

   Prüfen Sie die data-attribute auf Schreibfehler</li>

  </ul>
