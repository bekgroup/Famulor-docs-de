Der System-Prompt ist die wichtigste Komponente Ihres KI-Assistenten. Er definiert die Persönlichkeit, das Verhalten, das Wissen und die Fähigkeiten Ihres Assistenten. Betrachten Sie ihn als kombiniertes "Gehirn" und "Schulungsmanual".

## Schnellstart mit Vorlagen

Um schnell zu beginnen:

1. Gehen Sie zu Ihren Assistenten-Einstellungen
2. Finden Sie das System-Prompt-Feld
3. Klicken Sie auf den "Vorlagen"-Button in der Nähe
4. Wählen Sie eine Vorlage, die zu Ihrem Anwendungsfall passt
5. Passen Sie sie an Ihre Bedürfnisse an

## Sprachunterstützung

Ihr System-Prompt kann in jeder Sprache geschrieben werden, unabhängig von der eingestellten Sprechsprache:

- Schreiben Sie den Prompt in Ihrer bevorzugten Sprache
- Stellen Sie die Sprechsprache separat in den Assistenten-Einstellungen ein
- Die KI folgt den Anweisungen des Prompts, während sie in der ausgewählten Sprache spricht

Zum Beispiel:

- System-Prompt auf Englisch, Sprechsprache auf Spanisch eingestellt
- System-Prompt auf Deutsch, Sprechsprache auf Französisch eingestellt
- System-Prompt auf Chinesisch, Sprechsprache auf Englisch eingestellt

## Warum sie wichtig sind

Ihr System-Prompt:

- Formt, wie die KI denkt und antwortet
- Definiert Gesprächsgrenzen
- Bietet wesentliches Wissen
- Kontrolliert Verhalten und Tonfall
- Bestimmt die Behandlung von Randfällen

## Wichtige Komponenten

Ein guter System-Prompt sollte enthalten:

### 1. Rolle & Identität

```
Sie sind ein professioneller Vertriebsmitarbeiter für [Unternehmen]. Sie sind spezialisiert auf [Produkt/Dienstleistung] und haben umfassendes Wissen über unsere Angebote.
```

### 2. Gesprächsstil

```
Behalten Sie einen freundlichen, professionellen Tonfall bei. Verwenden Sie klare, prägnante Sprache. Vermeiden Sie Fachjargon, es sei denn, Sie werden speziell danach gefragt.
```

### 3. Wichtige Informationen

```
Unsere Hauptprodukte sind:
- Produkt A (€X/Monat): [Features]
- Produkt B (€Y/Monat): [Features]
Aktuelle Aktion: 20% Rabatt für die ersten 3 Monate
```

### 4. Verhaltensrichtlinien

```
- Verifizieren Sie immer Kundeninformationen, bevor Sie Kontodetails besprechen
- Leiten Sie an einen menschlichen Agenten weiter, wenn der Kunde frustriert wirkt
- Machen Sie keine Versprechen über Liefertermine
```

### 5. Antwortrahmen

```
Wenn nach Preisen gefragt wird:
1. Verstehen Sie zuerst ihre Bedürfnisse
2. Präsentieren Sie relevante Paketoptionen
3. Erklären Sie die Wertversprechung
4. Teilen Sie anwendbare Rabatte mit
```

## Best Practices

1. **Seien Sie spezifisch**

   - Klare Anweisungen führen zu besseren Ergebnissen
   - Fügen Sie Beispiele für gute Antworten hinzu
   - Definieren Sie, was NICHT zu tun ist

2. **Struktur ist wichtig**

   - Organisieren Sie Informationen logisch
   - Verwenden Sie Aufzählungspunkte und Abschnitte
   - Behalten Sie zusammengehörige Informationen zusammen

3. **Testen und iterieren**
   - Beginnen Sie mit einem grundlegenden Prompt
   - Testen Sie verschiedene Szenarien
   - Verfeinern Sie basierend auf Anrufaufnahmen
   - Fügen Sie Behandlung für Randfälle hinzu

## Häufige Fehler

- **Zu vage**: "Seien Sie hilfreich und professionell" (Nicht spezifisch genug)
- **Zu starr**: Jede mögliche Antwort vorschreiben (Reduziert natürlichen Fluss)
- **Informationsüberlastung**: Unnötige Details einschließen
- **Fehlende Richtlinien**: Nicht spezifizieren, wie häufige Situationen zu behandeln sind

## Beispielstruktur

```
# Rolle und Zweck
[Definieren Sie, wer die KI ist und ihre Hauptziele]

# Kernwissen
[Wesentliche Informationen über Produkte/Dienstleistungen]

# Gesprächsrichtlinien
[Wie mit Kunden zu interagieren ist]

# Antwortmuster
[Wie spezifische Situationen zu behandeln sind]

# Einschränkungen und Grenzen
[Was die KI NICHT tun sollte]
```

## Ihren Prompt testen

1. Machen Sie Testanrufe, die abdecken:

   - Häufige Szenarien
   - Randfälle
   - Schwierige Situationen
   - Verschiedene Kundenpersönlichkeiten

2. Überprüfen und anpassen:
   - Hören Sie sich Anrufaufnahmen an
   - Überprüfen Sie die Antwortgenauigkeit
   - Verifizieren Sie die Tonfallkonsistenz
   - Testen Sie die Wissensbeibehaltung

## Optimierungstipps

- Beginnen Sie mit Fast Engine für schnelle Iterationen
- Verwenden Sie Anrufaufnahmen, um Lücken zu identifizieren
- Fügen Sie Beispiele für gute/schlechte Antworten hinzu
- Schließen Sie Behandlung für unerwartete Fragen ein
