# Famulor Automation Platform - Integrations C-Z

Diese Seite setzt die umfassende Integrations-Datenbank fort und umfasst alle Integrationen von C bis Z mit detaillierten Informationen zu verfügbaren Aktionen und Auslösern.

## Integrations-Kategorien C-Z

### **C**

---
**Cal.com**
* **Beschreibung:** Open-Source-Terminplanungsplattform für Meetings und Events
* **Aktionen:**
    * **Create Booking:** Einen neuen Termin erstellen
    * **Update Booking:** Einen bestehenden Termin aktualisieren
    * **Cancel Booking:** Einen Termin stornieren
    * **Get Available Slots:** Verfügbare Terminslots abrufen
    * **Create Event Type:** Einen neuen Ereignistyp erstellen
    * **Get User Schedule:** Den Zeitplan eines Benutzers abrufen
* **Auslöser:**
    * **Booking Created:** Wird ausgelöst, wenn ein neuer Termin gebucht wird
    * **Booking Cancelled:** Wird ausgelöst, wenn ein Termin storniert wird
    * **Booking Rescheduled:** Wird ausgelöst, wenn ein Termin verschoben wird

---
**Calendly**
* **Beschreibung:** Terminplanungsplattform, die das Buchen von Meetings vereinfacht
* **Aktionen:**
    * **Create Event:** Ein neues Ereignis erstellen
    * **Update Event:** Ein bestehendes Ereignis aktualisieren
    * **Cancel Event:** Ein Ereignis stornieren
    * **Get User Info:** Benutzerinformationen abrufen
    * **List Events:** Alle geplanten Ereignisse auflisten
    * **Get Availability:** Verfügbare Zeiten abrufen
* **Auslöser:**
    * **Event Scheduled:** Wird ausgelöst, wenn ein neues Event geplant wird
    * **Event Cancelled:** Wird ausgelöst, wenn ein Event storniert wird
    * **Invitee Created:** Wird ausgelöst, wenn ein neuer Teilnehmer hinzugefügt wird

    * **Unsubscribe Contact:** Einen Kontakt abmelden
    * **Create Campaign:** Eine neue E-Mail-Kampagne erstellen
    * **Send Campaign:** Eine Kampagne versenden
    * **Get Campaign Stats:** Kampagnenstatistiken abrufen

---
**ClickUp**
* **Beschreibung:** All-in-One-Produktivitäts- und Projektmanagement-Platform
* **Aktionen:**
    * **Create Task:** Eine neue Aufgabe erstellen
    * **Update Task:** Eine Aufgabe aktualisieren
    * **Create List:** Eine neue Liste erstellen
    * **Add Comment:** Einen Kommentar hinzufügen
    * **Create Space:** Einen neuen Arbeitsbereich erstellen
    * **Track Time:** Zeit für Aufgaben erfassen
    * **Set Due Date:** Fälligkeitsdatum festlegen
* **Auslöser:**
    * **Task Created:** Wird ausgelöst, wenn eine neue Aufgabe erstellt wird
    * **Task Updated:** Wird ausgelöst, wenn eine Aufgabe aktualisiert wird
    * **Task Status Changed:** Wird ausgelöst, wenn sich der Aufgabenstatus ändert

---
**Clockodo**
* **Beschreibung:** Zeiterfassungs- und Projektmanagement-Software
* **Aktionen:**
    * **Start Timer:** Eine Zeiterfassung starten
    * **Stop Timer:** Eine Zeiterfassung beenden
    * **Create Entry:** Einen manuellen Zeiteintrag erstellen
    * **Update Entry:** Einen Zeiteintrag aktualisieren
    * **Generate Report:** Zeiterfassungsberichte erstellen
    * **Get Projects:** Alle Projekte abrufen

---
**ConvertKit**
* **Beschreibung:** E-Mail-Marketing-Platform für Content-Creator
* **Aktionen:**
    * **Add Subscriber:** Abonnenten hinzufügen
    * **Remove Subscriber:** Abonnenten entfernen
