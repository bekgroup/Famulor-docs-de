# Microsoft Dynamics 365 Integration Template

## Tool 1: Produktsuche

### Konfiguration im Famulor Interface

<Tabs>
  <Tab title="Werkzeugdetails">
    | **Name*** | `Dynamics 365 Produktsuche` |
    | **Funktionsname*** | `search_dynamics_products` |
    | **HTTP-Methode** | `GET` |
    | **Endpoint*** | `https://{{DYNAMICS_INSTANCE}}.crm.dynamics.com/api/data/v9.2/products` |
  </Tab>
  
  <Tab title="Query-Parameter">
    ```json
    {
      "$filter": "contains(name, '{search_term}') or productnumber eq '{product_number}'",
      "$select": "name,productnumber,price,quantityonhand,description",
      "$top": "10"
    }
    ```
  </Tab>
</Tabs>

**Success-Template**: `"Gefunden: {{products[0].name}} ({{products[0].productnumber}}), Preis: {{products[0].price}}€, Lagerbestand: {{products[0].quantityonhand}} Stück"`

## Tool 2: Auftrag anlegen

### Konfiguration im Famulor Interface

<Tabs>
  <Tab title="Werkzeugdetails">
    | **Name*** | `Dynamics 365 Auftrag anlegen` |
    | **Funktionsname*** | `create_dynamics_order` |
    | **HTTP-Methode** | `POST` |
    | **Endpoint*** | `https://{{DYNAMICS_INSTANCE}}.crm.dynamics.com/api/data/v9.2/salesorders` |
  </Tab>
</Tabs>

**Success-Template**: `"Auftrag wurde angelegt. Auftragsnummer: {{orderNumber}}. Lieferung geplant für {{delivery_date}}."`
