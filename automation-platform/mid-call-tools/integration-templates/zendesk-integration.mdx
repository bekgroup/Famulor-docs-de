# Zendesk Integration Template

## Konfiguration im Famulor Interface

<Tabs>
  <Tab title="Werkzeugdetails">
    | **Name*** | `Zendesk Ticket erstellen` |
    | **Funktionsname*** | `create_zendesk_ticket` |
    | **HTTP-Methode** | `POST` |
    | **Endpoint*** | `https://{{ZENDESK_SUBDOMAIN}}.zendesk.com/api/v2/tickets` |
    | **Timeout (ms)** | `5000` |
  </Tab>
  
  <Tab title="Request Body">
    ```json
    {
      "ticket": {
        "subject": "{subject}",
        "comment": {
          "body": "{description}"
        },
        "priority": "{priority}",
        "requester": {
          "name": "{requester_name}",
          "email": "{requester_email}"
        },
        "tags": "{tags}"
      }
    }
    ```
  </Tab>
</Tabs>

### Parameter-Schema

```json
{
  "type": "object",
  "properties": {
    "subject": {
      "type": "string",
      "description": "Ticket-Betreff"
    },
    "description": {
      "type": "string",
      "description": "Problembeschreibung"
    },
    "priority": {
      "type": "string",
      "enum": ["low", "normal", "high", "urgent"],
      "description": "Priorität"
    },
    "requester_name": {
      "type": "string",
      "description": "Name des Anfragenden"
    },
    "requester_email": {
      "type": "string",
      "format": "email", 
      "description": "E-Mail des Anfragenden"
    },
    "tags": {
      "type": "array",
      "description": "Tags für das Ticket"
    }
  },
  "required": ["subject", "description", "requester_email"]
}
```

**Agent-Message**: `"Ich erstelle ein Support-Ticket für {{requester_name}}..."`

**Success-Template**: `"Ticket wurde erstellt. Ticket-ID: {{response.ticket.id}}"`
