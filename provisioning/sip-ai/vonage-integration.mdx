# Vonage Integration

<Note>
  Eine Schritt-für-Schritt-Anleitung zur Verbindung mit Ihrem Vonage-Konto über SIP-Trunking mit Famulor.
</Note>

## Übersicht

<Card
  title="Was ist Vonage?"
  icon="building"
  color="blue">
  <p>Vonage ist ein führender Anbieter für Cloud-Kommunikationslösungen. Mit der SIP-Trunking-Integration können Sie Ihre bestehenden Vonage-Telefonnummern mit Famulor verbinden und von der KI-gestützten Anrufverwaltung profitieren.</p>
</Card>

## Schritt 1: Elastic SIP Trunking erstellen

<Card
  title="SIP-Trunk in Vonage einrichten"
  icon="settings"
  color="green">
  <ol>
    <li>Navigieren Sie zum <strong>SIP-Bereich</strong> im Vonage-Dashboard</li>
    <li>Wählen Sie <strong>"Other"</strong> als Anbieter aus</li>
    <li>Folgen Sie den Anweisungen Schritt für Schritt</li>
  </ol>
</Card>

### Termination einrichten (für ausgehende Anrufe)

<Card
  title="Wichtige Informationen notieren"
  icon="edit"
  color="orange">
  <p>Bei der Einrichtung der Termination (ausgehende Anrufe) sind folgende Informationen wichtig:</p>
  <ul>
    <li><strong>Termination SIP URI</strong></li>
    <li><strong>Benutzername</strong></li>
    <li><strong>Passwort</strong></li>
  </ul>
  <p><strong>Wichtig:</strong> Notieren Sie sich diese Informationen, da Sie sie in den späteren Schritten benötigen werden.</p>
</Card>

### Origination einrichten (für eingehende Anrufe)

<Card
  title="Famulor SIP-Server konfigurieren"
  icon="server"
  color="purple">
  <p>Hier geben Sie die Famulor SIP-Server-Adresse als Origination SIP URI an:</p>
  <CodeGroup>
    <CodeGroupItem title="SIP-Server">
      ```bash
      sip:ihre-unique-id.sip.livekit.cloud
      ```
    </CodeGroupItem>
  </CodeGroup>
  
  <Note>
    **Wichtig:** Die spezifische Famulor SIP-Adresse finden Sie in Ihrem Famulor-Dashboard unter "Telefonnummern" → "SIP-Trunk integrieren" → "Unsere SIP-Adresse".
  </Note>
</Card>

## Schritt 2: Nummern zu Elastic SIP Trunking verschieben

<Card
  title="Nummern verknüpfen"
  icon="phone"
  color="blue">
  <p>Nachdem Sie den Elastic SIP-Trunk erstellt haben, müssen Sie:</p>
  <ul>
    <li>Neue Nummern kaufen <strong>ODER</strong></li>
    <li>Bestehende Nummern mit diesem Trunk verknüpfen</li>
  </ul>
</Card>

## Schritt 3: Nummern in Famulor importieren

<Card
  title="Integration in Famulor abschließen"
  icon="download"
  color="green">
  <p>Jetzt, da die Nummer mit Ihrem Elastic SIP-Trunking eingerichtet ist, müssen Sie sie in Famulor importieren, damit wir wissen, wie wir die Anrufe routen können.</p>
</Card>

### Benötigte Informationen

<Card
  title="Vonage-Daten eingeben"
  icon="database"
  color="red">
  <p>Geben Sie hier die folgenden Informationen ein, die Sie in Schritt 1 eingerichtet haben:</p>
  <ul>
    <li><strong>Termination SIP URI</strong> aus Schritt 1</li>
    <li><strong>Benutzername</strong> aus Schritt 1</li>
    <li><strong>Passwort</strong> aus Schritt 1</li>
  </ul>
</Card>

### Programmgesteuerter Import

<Card
  title="API-Import verfügbar"
  icon="code"
  color="indigo">
  <p>Sie können Nummern auch programmgesteuert über die <strong>Import Number API</strong> importieren.</p>
</Card>

## Nach dem Import

<Card
  title="Vollständige Funktionalität"
  icon="check-circle"
  color="green">
  <p>Sobald die Nummer importiert ist, können Sie:</p>
  <ul>
    <li>Anrufe mit dieser Nummer tätigen und empfangen</li>
    <li>Die Nummer wird in Ihrem Dashboard angezeigt</li>
    <li>Sie können Assistenten an die Nummer binden</li>
    <li>Direkt vom Dashboard aus Anrufe tätigen</li>
  </ul>
</Card>

  title="Weiterführende Informationen"
  icon="book-open"
  color="blue">

</Card>

## Häufige Probleme und Lösungen

<Card
  title="Troubleshooting"
  icon="help-circle"
  color="orange">
  <ul>
    <li><strong>Verbindungsprobleme:</strong> Überprüfen Sie die SIP-URI und Anmeldedaten</li>
    <li><strong>Anrufe funktionieren nicht:</strong> Stellen Sie sicher, dass der Trunk aktiviert ist</li>
    <li><strong>Eingehende Anrufe fehlen:</strong> Überprüfen Sie die Origination-Einstellungen</li>
    <li><strong>Ausgehende Anrufe fehlen:</strong> Überprüfen Sie die Termination-Einstellungen</li>
  </ul>
</Card>

## Einrichtung in Famulor

<Card
  title="Schritte zur Einrichtung"
  icon="settings"
  color="purple">
  <ol>
    <li>Gehen Sie zu <strong>Telefonnummern</strong> → <strong>SIP-Trunk integrieren</strong></li>
    <li>Wählen Sie <strong>"Vonage"</strong> als Anbieter</li>
    <li>Geben Sie die <strong>Termination SIP URI</strong> ein</li>
    <li>Geben Sie den <strong>Benutzernamen</strong> ein</li>
    <li>Geben Sie das <strong>Passwort</strong> ein</li>
    <li>Klicken Sie auf <strong>"Speichern"</strong></li>
    <li>Testen Sie die Verbindung mit einem Anruf</li>
  </ol>
</Card>
