# Placetel-Limitationen

<Note>
  Placetel unterstützt derzeit leider keine statischen SIP Trunks für die Integration mit Famulor.
</Note>

## Warum funktioniert die Integration derzeit nicht?

<Card
  title="Aktuelle technische Limitation"
  icon="alert-triangle"
  color="red">
  <strong>Placetel unterstützt derzeit leider keine statischen SIP Trunks.</strong>
  
  Dies bedeutet, dass eine direkte SIP-Integration zwischen Placetel und Famulor derzeit technisch nicht möglich ist.
</Card>

<Card
  title="Direkt beim Anbieter nachfragen"
  icon="phone"
  color="blue">
  <strong>Wichtig:</strong> Fragen Sie direkt bei Placetel nach, ob sie statische SIP Trunks anbieten können!
  
  <ul>
    <li>Viele Anbieter entwickeln ihre Services kontinuierlich weiter</li>
    <li>Möglicherweise ist die Funktion bereits verfügbar oder in Planung</li>
    <li>Direkte Nachfrage kann zu individuellen Lösungen führen</li>
  </ul>
</Card>

## Was bedeutet das für Sie?

<CardGroup cols={2}>
  <Card
    title="Eingeschränkte Funktionalität"
    icon="phone-x"
    color="orange">
    <ul>
      <li>Keine direkte SIP-Trunk-Integration möglich</li>
      <li>Keine automatische Weiterleitung von Anrufen an KI-Assistenten</li>
      <li>Eingeschränkte Automatisierungsmöglichkeiten</li>
    </ul>
  </Card>
  
  <Card
    title="Alternative Lösungen"
    icon="lightbulb"
    color="blue">
    <ul>
      <li>Manuelle Weiterleitung von Anrufen</li>
      <li>Verwendung von Famulor-Nummern</li>
      <li>Call-Forwarding über Placetel-Dashboard</li>
    </ul>
  </Card>
</CardGroup>

## Unsere Empfehlung

<Card
  title="Einfache Alternative"
  icon="star"
  color="green">
  <strong>Sie können direkt eine Nummer bei uns kaufen und nutzen!</strong>
  
  <ul>
    <li><strong>Inbound:</strong> Leiten Sie einfach Ihre aktuelle Placetel-Nummer an die Famulor-Nummer weiter</li>
    <li><strong>Outbound:</strong> Verifizieren Sie die Nummer bei "Telefonnummern" und telefonieren Sie direkt damit raus</li>
  </ul>
  
  Diese Lösung bietet Ihnen:
  
  <ul>
    <li>Vollständige KI-Assistenten-Funktionalität</li>
    <li>Automatisierte Anrufbehandlung</li>
    <li>Erweiterte Telefonie-Features</li>
    <li>Nahtlose Integration mit der Famulor-Plattform</li>
    <li>Einfache Einrichtung über Call-Forwarding</li>
  </ul>
</Card>
