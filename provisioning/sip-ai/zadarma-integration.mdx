## Was ist Zadarma-Integration?

Die Zadarma-Integration ermöglicht es Ihnen, Ihr Zadarma-Telefonsystem mit Famulor zu verbinden. Zadarma bietet flexible SIP-Trunking-Lösungen für die Telefonie-Integration.

<Steps>
  <Step title="Bestehende Nummern nutzen">
    Ihre bestehenden Zadarma-Telefonnummern mit unseren KI-Assistenten nutzen
  </Step>
  <Step title="Infrastruktur beibehalten">
    Zadarma's robuste Infrastruktur und globale Reichweite beibehalten
  </Step>
  <Step title="Anrufe verwalten">
    Anrufe über Ihr Zadarma-Setup tätigen und empfangen
  </Step>
  <Step title="Kontrolle behalten">
    Die vollständige Kontrolle über Ihre Nummern behalten
  </Step>
  <Step title="Kosten sparen">
    Von Zadarma's wettbewerbsfähigen Tarifen und regionalen Endpunkten profitieren
  </Step>
</Steps>

## Einrichtung Ihrer Zadarma-Integration

<Steps>
  <Step title="Informationen sammeln">
    <CardGroup cols={2}>
      <Card
        title="PBX-Erweiterung"
        icon="phone"
        color="blue">
        <strong>Gehen Sie zu "Meine Telefonanlage" → "Erweiterungen" auf der Hauptseite.</strong>
        
        ![Zadarma Meine Telefonanlage Erweiterungen](/provisioning/sip-ai/zadarma-mypbx-extensions.png)
        
        Finden Sie folgende Informationen:
        <CodeGroup>
          <CodeGroupItem title="Beispiel">
            ```bash
            Erweiterungsnummer: z.B. 1000
            Server-Adresse: z.B. pbx.zadarma.com
            Benutzername: Ihr Erweiterungs-Benutzername
            Passwort: Ihr Erweiterungs-Passwort
            ```
          </CodeGroupItem>
        </CodeGroup>
        
        ![Zadarma Extension Settings](/provisioning/sip-ai/zadarma-switch-extension-settings.PNG)
      </Card>
      <Card
        title="SIP-Verbindung"
        icon="server"
        color="green">
        <strong>Gehen Sie zu "Konfiguration" → "SIP-Einstellungen" auf der Hauptseite.</strong>
        
        ![Zadarma SIP Settings](/provisioning/sip-ai/zadarma-sip-settings.png)
        
        Finden Sie folgende Informationen:
        <CodeGroup>
          <CodeGroupItem title="Beispiel">
            ```bash
            Telefonnummer: z.B. +1 555 0123
            SIP-Server: z.B. sip.zadarma.com
            Benutzername: Von Ihrer Telefongesellschaft bereitgestellt
            Passwort: Von Ihrer Telefongesellschaft bereitgestellt
            ```
          </CodeGroupItem>
        </CodeGroup>
        
        ![Zadarma SIP Connection Settings](/provisioning/sip-ai/zadarma-sip-connection-settings.png)
      </Card>
    </CardGroup>
    
    <Card
      title="IP-Autorisierungstyp"
      icon="shield"
      color="orange">
      <strong>Für IP-Autorisierungstyp:</strong>
      
      1. Gehen Sie zu "Dienste" → "Nummer verbinden" auf der Hauptseite
      
      ![Zadarma Services](/provisioning/sip-ai/zadarma-services.png)
      
      2. Gehen Sie zu "Aktive Nummern"
      
      ![Zadarma Active Numbers](/provisioning/sip-ai/zadarma-active-numbers.png)
      3. Klicken Sie auf die Nummer, die Sie verbinden möchten, und drücken Sie den Einstellungs-Button
      
      ![Zadarma Settings Button](/provisioning/sip-ai/zadarma-settings-button.png)
      
      4. Klicken Sie auf "Externe Dienste" und aktivieren Sie dann die Option für SIP URI
      
      ![Zadarma External Services](/provisioning/sip-ai/zadarma-external-services.png)
      5. Fügen Sie `sip:extension@ihre-unique-id.sip.livekit.cloud` hinzu (dies ist die Famulor SIP-Server-Adresse)
      
      <Note>
        Sie benötigen diese IPs, die Sie in Schritt 2 für den eingehenden Autorisierungstyp eingeben werden.
      </Note>
    </Card>
    
    <Note>
      **Wichtig:** Die spezifische Famulor SIP-Adresse finden Sie in Ihrem Famulor-Dashboard unter "Telefonnummern" → "SIP-Trunk integrieren" → "Unsere SIP-Adresse".
    </Note>
  </Step>
  
  <Step title="Nummern zu Famulor importieren">
    <Card
      title="Import-Prozess"
      icon="download"
      color="blue">
      Sie müssen zu Famulor importieren.
      
      1. Gehen Sie zu Ihrem Famulor-Dashboard
      2. Navigieren Sie zu "Ihre Telefonnummern"
      3. Klicken Sie auf "SIP-Trunk integrieren"
    </Card>
    
    <CardGroup cols={2}>
      <Card
        title="PBX-Erweiterung Details"
        icon="settings"
        color="purple">
        Füllen Sie Ihre Details basierend auf Ihrem Integrationstyp aus.
      </Card>
      <Card
        title="SIP-Verbindung Details"
        icon="link"
        color="green">
        Füllen Sie Ihre Details basierend auf Ihrem Integrationstyp aus.
      </Card>
    </CardGroup>
    
    <Card
      title="Server-Adresse"
      icon="server"
      color="blue">
      Hier geben Sie die Termination SIP URI an, die Sie in Schritt 1 eingerichtet haben.
      
      <CodeGroup>
        <CodeGroupItem title="Beispiel">
          ```bash
          Server-Adresse: z.B. pbx.zadarma.com / sip.zadarma.com basierend auf Ihrem Integrationstyp
          ```
        </CodeGroupItem>
      </CodeGroup>
    </Card>
    
    <Card
      title="Autorisierungstyp"
      icon="shield"
      color="orange">
      Wählen Sie hier den Autorisierungstyp aus.
      
      Fügen Sie die IPs von Zadarma hinzu (Schritt 1).
    </Card>
    
    <Card
      title="Land auswählen"
      icon="globe"
      color="green">
      Wählen Sie das Land aus, in dem sich Ihr SIP-Trunk befindet.
    </Card>
  </Step>
  
  <Step title="Einrichtung verifizieren">
    <Card
      title="Verifikationsschritte"
      icon="check-circle"
      color="green">
      Nach der Konfiguration:
      
      1. Tätigen Sie einen Testanruf an Ihre Nummer/Erweiterung
      2. Überprüfen Sie, dass der KI-Assistent antwortet
      3. Verwenden Sie die Testanruf-Funktion, um ausgehende Anrufe zu verifizieren
    </Card>
  </Step>
</Steps>

## Sicherheitsanforderungen

<Card
  title="SIP-Anmeldedaten Sicherheit"
  icon="shield"
  color="red">
  Ihre SIP-Anmeldedaten müssen sicher sein:
  
  - **Benutzername:** Ihre Erweiterungsnummer oder von Zadarma bereitgestellter Benutzername
  - **Passwort muss haben:**
    - 12 oder mehr Zeichen
    - Zahlen
    - Groß- und Kleinbuchstaben
    - Sonderzeichen empfohlen
</Card>

## Häufige Probleme

<CardGroup cols={2}>
  <Card
    title="Ausgehende Anrufe funktionieren nicht"
    icon="phone-x"
    color="red">
    <strong>Nach der Verbindung funktioniert der eingehende Anruf, aber der ausgehende funktioniert nicht?</strong>
    
    <Tabs>
      <TabItem title="Termination SIP URI">
        - Wenn es ein Leerzeichen gibt, entfernen Sie es bitte
        - Verwenden Sie eine lokalisierte Termination URI, die sich in Ihrer Nähe befindet
        - Schauen Sie sich diese Dokumentation an, um eine auszuwählen
      </TabItem>
      <TabItem title="Benutzername & Anmeldedaten">
        - Stellen Sie sicher, dass Sie den richtigen Benutzernamen und die richtigen Anmeldedaten eingegeben haben
        - Beachten Sie, dass der Benutzername nicht der freundliche Name ist, der in der Anmeldedatenliste angezeigt wird
        - Der freundliche Name unterscheidet sich vom Benutzernamen - überprüfen Sie dies doppelt
      </TabItem>
    </Tabs>
  </Card>
  
  <Card
    title="Internationale Länder einrichten"
    icon="globe"
    color="blue">
    <strong>Wie richte ich das Wählen in internationale Länder ein?</strong>
    
    1. Suchen Sie nach "geo", um die "Voice Geographic Permissions"-Einstellung zu finden
    2. Wählen Sie "Elastic SIP Trunking" im Selektor aus
    3. Wählen Sie die Länder aus, in die Sie wählen möchten
  </Card>
</CardGroup>
