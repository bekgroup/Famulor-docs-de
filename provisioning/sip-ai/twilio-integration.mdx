# Twilio-Integration

Eine Schritt-für-Schritt-Anleitung zur Verbindung Ihres Twilio-Kontos über SIP-Trunking mit Famulor.

## Schritt 1: Elastic SIP Trunking erstellen

Erstellen Sie den Trunk, geben Sie ihm einen Namen und aktivieren Sie die allgemeinen Einstellungen.

![Twilio Create Elastic SIP Trunking](/provisioning/sip-ai/twillio-create-elastic-sip1.jpeg)

### Termination einrichten (für ausgehende Anrufe)

Die **Termination SIP URI** ist hier wichtig - wir werden sie in späteren Schritten verwenden. 

<Tip>
Verwenden Sie eine **lokalisierte Termination URI**, die sich in Ihrer Region befindet. Sie können Ihre lokalisierten URIs in der Twilio-Konsole erweitern und anzeigen.
</Tip>

![Twilio Setup Termination](/provisioning/sip-ai/twillio-setup-termination1.jpeg)

#### Zugriffskontrolle konfigurieren

Damit Ihr Elastic SIP Trunk unsere ausgehenden Anfragen akzeptiert, müssen Sie **entweder** IP-Adressen whitelisten **oder** eine Authentifizierung mit Benutzername und Passwort erstellen.

<Note>
Da der Famulor SIP-Server derzeit keine statische IP hat, müssen Sie, wenn Sie sich für die **IP-Route** entscheiden, alle IP-Adressen im Bereich whitelisten.
</Note>

**IP-Adressen für Whitelist:**
```bash
54.244.51.0
54.244.51.1  
54.244.51.2
54.244.51.3
54.172.60.0
54.172.60.1
54.172.60.2
54.172.60.3
```

![Twilio New Access Control](/provisioning/sip-ai/twillio-new-access-control.jpeg)

![Twilio New Access Control Settings](/provisioning/sip-ai/twillio-new-access-control2.jpeg)

### Origination einrichten (für eingehende Anrufe)

Hier geben Sie die **Famulor SIP-Server-Adresse** als Origination SIP URI an:

```bash
sip:5wed79aeavr.sip.livekit.cloud
```

<Warning>
**Wichtig:** Die spezifische Famulor SIP-Adresse finden Sie in Ihrem Famulor-Dashboard unter:  
**"Ihre Telefonnummern"** → **"SIP-Trunk integrieren"** → **"Unsere SIP-Adresse"**
</Warning>

![Twilio Setup Origination](/provisioning/sip-ai/twillio-setup-origination1.jpeg)

## Schritt 2: Nummern zu Elastic SIP Trunking verschieben

Sie haben den Elastic SIP Trunk erstellt, jetzt müssen Sie Nummern kaufen oder bestehende Nummern zu diesem Trunk verschieben.

![Twilio Move Numbers](/provisioning/sip-ai/twillio-move-numbers1.jpeg)

---

## Schritt 3: Nummern zu Famulor importieren

Jetzt, da die Nummer mit Ihrem Elastic SIP Trunking eingerichtet ist, müssen Sie die Nummer zu Famulor importieren, damit wir wissen, wie wir den Anruf routen können.

1. Gehen Sie zu Ihrem **Famulor-Dashboard**
2. Navigieren Sie zu **"Ihre Telefonnummern"**
3. Klicken Sie auf **"SIP-Trunk integrieren"**

Füllen Sie Ihre Details basierend auf Ihrem Integrationstyp aus:

```bash
Telefonnummer: z.B. +1 555 0123
Benutzername: Ihr Erweiterungs-Benutzername
Passwort: Ihr Erweiterungs-Passwort
```

Hier geben Sie die **Termination SIP URI** an, die Sie in Schritt 1 eingerichtet haben:

```bash
Server-Adresse: z.B. xxx.pstn.twilio.com
```

Wählen Sie den **Autorisierungstyp** aus, den Sie in Schritt 1 gewählt haben.

Wenn Sie den **IP-Typ** gewählt haben, fügen Sie die IPs hinzu:

```bash
54.244.51.0
54.244.51.1  
54.244.51.2
54.244.51.3
54.172.60.0
54.172.60.1
54.172.60.2
54.172.60.3
```

Wählen Sie das **Land** aus, in dem sich Ihr SIP-Trunk befindet.

---

## Schritt 4: Einrichtung verifizieren

Nach der Konfiguration:

- Tätigen Sie einen Testanruf an Ihre Nummer/Erweiterung
- Überprüfen Sie, dass der KI-Assistent antwortet
- Verwenden Sie die Testanruf-Funktion, um ausgehende Anrufe zu verifizieren

---

## Sicherheitsanforderungen

Ihre SIP-Anmeldedaten müssen sicher sein:

- **Benutzername:** Ihre Erweiterungsnummer oder von Twilio bereitgestellter Benutzername
- **Passwort muss haben:**
  - 12 oder mehr Zeichen
  - Zahlen
  - Groß- und Kleinbuchstaben
  - Sonderzeichen empfohlen

---

## Häufige Probleme

### Nach der Verbindung funktioniert der eingehende Anruf, aber der ausgehende funktioniert nicht?

**Überprüfen Sie Ihre Termination SIP URI:**
- Wenn es ein Leerzeichen gibt, entfernen Sie es bitte
- Verwenden Sie eine lokalisierte Termination URI, die sich in Ihrer Region befindet
- Schauen Sie sich [diese Dokumentation](https://www.twilio.com/docs/sip-trunking/termination-sip-domain) an, um eine auszuwählen

**Überprüfen Sie Ihren Benutzernamen & Anmeldedaten:**
- Stellen Sie sicher, dass Sie den richtigen Benutzernamen und die richtigen Anmeldedaten eingegeben haben
- Beachten Sie, dass der **Benutzername** nicht der freundliche Name ist, der in der Anmeldedatenliste angezeigt wird
- Der freundliche Name unterscheidet sich vom Benutzernamen - überprüfen Sie dies doppelt

![Twilio Outbound Not Working](/provisioning/sip-ai/twillio-outbound-not-work.png)

### Wie richte ich das Wählen in internationale Länder ein?

1. Suchen Sie nach **"geo"**, um die **"Voice Geographic Permissions"**-Einstellung zu finden
2. Wählen Sie **"Elastic SIP Trunking"** im Selektor aus
3. Wählen Sie die Länder aus, in die Sie wählen möchten

![Twilio Choose Elastic SIP Trunking](/provisioning/sip-ai/twillio-choose-elastic-sip-trunking.png)

![Twilio Setup Dialing International](/provisioning/sip-ai/twillio-setup-dialing-international.png)
