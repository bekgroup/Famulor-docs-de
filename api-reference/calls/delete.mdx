---
title: "Anruf löschen"
api: "DELETE https://app.famulor.de/api/user/calls/{call}"
icon: "phone-slash"
description: "Einen spezifischen Anrufdatensatz löschen"
---

Dieser Endpunkt ermöglicht es Ihnen, einen spezifischen Anrufdatensatz zu löschen, der dem authentifizierten Benutzer gehört.

### Path Parameter

<ParamField path="call" type="integer" required>
  Die eindeutige Kennung des zu löschenden Anrufs
</ParamField>

### Response

<ResponseField name="message" type="string">
  Bestätigungsnachricht, dass der Anruf erfolgreich gelöscht wurde
</ResponseField>

### Fehler-Responses

<ResponseField name="message" type="string">
  Fehlermeldung, wenn der Anruf nicht gefunden wurde oder nicht dem authentifizierten Benutzer gehört
</ResponseField>

### Hinweise

- Nur Anrufe, die zu Assistenten gehören, die dem authentifizierten Benutzer gehören, können gelöscht werden
- Sobald ein Anruf gelöscht wurde, kann er nicht wiederhergestellt werden
- Diese Aktion entfernt dauerhaft den Anrufdatensatz, einschließlich aller zugehörigen Transkripte, Aufzeichnungen und Metadaten
- Die Anrufaufzeichnungsdatei (falls vorhanden) wird ebenfalls aus dem Speicher gelöscht

<RequestExample>

```bash Anruf löschen
curl -X DELETE "https://app.famulor.de/api/user/calls/480336" \
  -H "Authorization: Bearer YOUR_API_TOKEN"
```

</RequestExample>

<ResponseExample>

```json 200 Success
{
  "message": "Call deleted successfully"
}
```

```json 404 Not Found
{
  "message": "Call not found or does not belong to the authenticated user"
}
```

</ResponseExample>
