---
title: "Anrufe auflisten"
api: "GET https://app.famulor.de/api/user/calls"
icon: "phone"
description: "Alle Anrufe für den authentifizierten Benutzer mit Filteroptionen auflisten"
---

Dieser Endpunkt ermöglicht es Ihnen, alle Anrufe aufzulisten, die dem authentifizierten Benutzer gehören, mit verschiedenen Filteroptionen.

### Query-Parameter

<ParamField query="status" type="string">
  Anrufe nach Status filtern. Mögliche Werte: `initiated`, `ringing`, `busy`, `in-progress`, `ended`, `completed`, `ended_by_customer`, `ended_by_assistant`, `no-answer`, `failed`
</ParamField>

<ParamField query="type" type="string">
  Anrufe nach Typ filtern. Mögliche Werte: `inbound`, `outbound`, `web`
</ParamField>

<ParamField query="phone_number" type="string">
  Anrufe nach Kunden-Telefonnummer filtern
</ParamField>

<ParamField query="assistant_id" type="integer">
  Anrufe nach Assistenten-ID filtern
</ParamField>

<ParamField query="campaign_id" type="integer">
  Anrufe nach Kampagnen-ID filtern
</ParamField>

<ParamField query="date_from" type="string">
  Anrufe ab diesem Datum filtern (YYYY-MM-DD Format)
</ParamField>

<ParamField query="date_to" type="string">
  Anrufe bis zu diesem Datum filtern (YYYY-MM-DD Format)
</ParamField>

<ParamField query="per_page" type="integer">
  Anzahl der Anrufe pro Seite (1-100, Standard: 15)
</ParamField>

<ParamField query="page" type="integer">
  Seitennummer (Standard: 1)
</ParamField>

### Antwort-Felder

<ResponseField name="data" type="array">
  Array der Anrufe

  <Expandable title="Anruf-Eigenschaften">
    <ResponseField name="id" type="integer">
      Die eindeutige Kennung des Anrufs
    </ResponseField>

    <ResponseField name="assistant_name" type="string">
      Der Name des Assistenten, der den Anruf bearbeitet hat
    </ResponseField>

    <ResponseField name="campaign_name" type="string">
      Der Name der Kampagne, zu der dieser Anruf gehört (falls zutreffend)
    </ResponseField>

    <ResponseField name="type" type="string">
      Der Typ des Anrufs (inbound, outbound oder web)
    </ResponseField>

    <ResponseField name="duration" type="integer">
      Die Dauer des Anrufs in Sekunden
    </ResponseField>

    <ResponseField name="assistant_phone_number" type="string">
      Die vom Assistenten verwendete Telefonnummer
    </ResponseField>

    <ResponseField name="client_phone_number" type="string">
      Die Telefonnummer des Kunden
    </ResponseField>

    <ResponseField name="status" type="string">
      Der aktuelle Status des Anrufs
    </ResponseField>

    <ResponseField name="transcript" type="array">
      Das Transkript der Anrufkonversation als Array von Objekten mit text, type, sender und timestamp
    </ResponseField>

    <ResponseField name="variables" type="object">
      Während des Anrufs gesammelte Variablen
    </ResponseField>

    <ResponseField name="evaluation" type="array">
      Bewertungsdaten für die Anrufleistung als Array von Objekten mit name, type, value und description
    </ResponseField>

    <ResponseField name="webhook_response" type="object">
      Antwort von konfigurierten Webhooks
    </ResponseField>

    <ResponseField name="carrier_cost" type="number">
      Die vom Anbieter berechneten Kosten für diesen Anruf
    </ResponseField>

    <ResponseField name="total_cost" type="number">
      Die Gesamtkosten des Anrufs inklusive aller Gebühren
    </ResponseField>

    <ResponseField name="answered_by" type="string">
      Wer den Anruf beantwortet hat (human, machine oder unknown)
    </ResponseField>

    <ResponseField name="recording_url" type="string">
      URL zur Anrufaufzeichnung (falls verfügbar und aktiviert)
    </ResponseField>

    <ResponseField name="created_at" type="string">
      Datum und Uhrzeit der Anruferstellung
    </ResponseField>

    <ResponseField name="updated_at" type="string">
      Datum und Uhrzeit der letzten Aktualisierung des Anrufs
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="current_page" type="integer">
  Die aktuelle Seitennummer
</ResponseField>

<ResponseField name="per_page" type="integer">
  Anzahl der Elemente pro Seite
</ResponseField>

<ResponseField name="total" type="integer">
  Gesamtanzahl der Anrufe, die den Kriterien entsprechen
</ResponseField>

<ResponseField name="last_page" type="integer">
  Die letzte Seitennummer
</ResponseField>

<RequestExample>

```bash Alle Anrufe abrufen
GET /user/calls
```

```bash Anrufe mit Filtern
GET /user/calls?status=completed&type=outbound&date_from=2024-01-01&per_page=25
```

```bash Anrufe einer bestimmten Kampagne
GET /user/calls?campaign_id=2324&page=2
```

</RequestExample>
