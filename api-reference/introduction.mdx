Willkommen zur Famulor API-Dokumentation. Unsere API ermöglicht es Ihnen, programmatisch mit der Famulor-Plattform zu interagieren, um Leads zu erstellen und zu verwalten, Anrufe zu planen und mehr.

## Basis-URL

Alle API-Anfragen sollten an die folgende Basis-URL gesendet werden:

```
https://app.famulor.de/api
```

## Kommende Funktionen

Wir arbeiten kontinuierlich an der Erweiterung unserer API. Eine der nächsten größeren Erweiterungen ist:

### 1. SIP-Trunk Import (End-to-End)
Wir planen die Einführung von API-Endpunkten für das Onboarding eigener SIP-Trunks inklusive der Konfiguration von SIP-Zugangsdaten, Adressen und Proxy-Einstellungen. Bis zur Fertigstellung erfolgt die Einrichtung eigener Trunks weiterhin manuell über unser Admin-Panel.
