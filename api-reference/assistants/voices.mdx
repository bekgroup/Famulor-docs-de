title: "Verfügbare Stimmen abrufen"
api: "GET https://app.famulor.de/api/user/assistants/voices"
icon: "microphone"
description: "Alle verfügbaren Stimmen für die Assistenten-Konfiguration abrufen"
---

Dieser Endpunkt gibt eine Liste aller verfügbaren Stimmen zurück, die beim Erstellen oder Aktualisieren von Assistenten verwendet werden können, mit optionaler Filterung nach Assistenten-Modus.

### Query-Parameter

<ParamField query="mode" type="string">
  Stimmen nach Assistenten-Modus filtern. Optionen: `pipeline`, `multimodal`
</ParamField>

### Antwort-Felder

Die API gibt ein direktes Array von verfügbaren Stimmen zurück:

<ResponseField name="array" type="array">
  Array der verfügbaren Stimmen

  <Expandable title="Stimmen-Eigenschaften">
    <ResponseField name="id" type="integer">
      Die eindeutige Kennung der Stimme
    </ResponseField>

    <ResponseField name="name" type="string">
      Der Anzeigename der Stimme
    </ResponseField>

    <ResponseField name="mode" type="string">
      Der Assistenten-Modus, mit dem diese Stimme kompatibel ist (pipeline oder multimodal)
    </ResponseField>
  </Expandable>
</ResponseField>

### Hinweise

- Verwenden Sie das `id`-Feld beim Erstellen oder Aktualisieren von Assistenten
- Das `mode`-Feld zeigt an, mit welcher Assistenten-Engine die Stimme kompatibel ist
- Filtern Sie nach Modus, um nur Stimmen zu erhalten, die mit Ihrem gewünschten Assistenten-Typ kompatibel sind
- Wenn kein Modus-Filter angegeben wird, werden alle verfügbaren Stimmen zurückgegeben
- Nur Stimmen, die öffentlich sind oder dem authentifizierten Benutzer gehören, werden zurückgegeben

<RequestExample>

```bash Alle verfügbaren Stimmen
curl -H "Authorization: Bearer YOUR_API_TOKEN" \
  "https://app.famulor.de/api/user/assistants/voices"
```

```bash Nur Pipeline-Stimmen
curl -H "Authorization: Bearer YOUR_API_TOKEN" \
  "https://app.famulor.de/api/user/assistants/voices?mode=pipeline"
```

```bash Nur Multimodal-Stimmen
curl -H "Authorization: Bearer YOUR_API_TOKEN" \
  "https://app.famulor.de/api/user/assistants/voices?mode=multimodal"
```

</RequestExample>
