description: "Alle verfügbaren Stimmen für die Assistenten-Konfiguration abrufen"
---

Dieser Endpunkt gibt eine Liste aller verfügbaren Stimmen zurück, die beim Erstellen oder Aktualisieren von Assistenten verwendet werden können, mit optionaler Filterung nach Assistenten-Modus und Sprache.

### Query-Parameter

<ParamField query="mode" type="string">
  Stimmen nach Assistenten-Modus filtern. Optionen: `pipeline`, `multimodal`, `dualplex`
</ParamField>

<ParamField query="language_id" type="integer">
  Stimmen nach Sprach-ID filtern. Verwenden Sie den Endpunkt [Sprachen abrufen](/api-reference/assistants/languages), um verfügbare Sprach-IDs zu erhalten.
</ParamField>

### Antwort-Felder

<ResponseField name="data" type="array">
  <Expandable title="Eigenschaften">
    <ResponseField name="id" type="integer">
      Die eindeutige Kennung der Stimme
    </ResponseField>

    <ResponseField name="name" type="string">
      Der Anzeigename der Stimme
    </ResponseField>

    <ResponseField name="language_id" type="integer">
      Die Sprach-ID, der diese Stimme zugeordnet ist
    </ResponseField>

    <ResponseField name="gender" type="string">
      Das Geschlecht der Stimme (male/female)
    </ResponseField>

    <ResponseField name="preview_url" type="string">
      URL zu einem Audio-Beispiel der Stimme. Kann verwendet werden, um Nutzern eine Vorschau der Stimme zu ermöglichen, bevor sie ausgewählt wird.
    </ResponseField>

    <ResponseField name="accent" type="string">
      Der Akzent der Stimme (z.B. „American“, „British“). Kann `null` sein.
    </ResponseField>

    <ResponseField name="age" type="string">
      Die Alterskategorie der Stimme (z.B. „Young“, „Middle aged“). Kann `null` sein.
    </ResponseField>

    <ResponseField name="synthesizer_provider_id" type="integer">
      Die TTS-Provider-ID, zu der diese Stimme gehört. Verwenden Sie den Endpunkt [Synthesizer-Provider abrufen](/api-reference/assistants/synthesizer-providers), um Providernamen aufzulösen. Kann `null` sein.
    </ResponseField>
  </Expandable>
</ResponseField>

### Hinweise

* Verwenden Sie das `id`-Feld als `voice_id` beim Erstellen oder Aktualisieren von Assistenten
* Filtern Sie nach `mode`, um nur Stimmen zu erhalten, die mit Ihrem gewünschten Assistenten-Typ kompatibel sind
* Filtern Sie nach `language_id`, um Stimmen für eine bestimmte Sprache zu erhalten
* Wenn keine Filter angegeben werden, werden alle verfügbaren Stimmen zurückgegeben
* Nur Stimmen, die öffentlich sind oder dem authentifizierten Benutzer gehören, werden zurückgegeben
