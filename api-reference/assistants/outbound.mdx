---
title: "Outbound-Assistenten abrufen"
api: "GET https://app.famulor.de/api/user/assistants/outbound"
icon: "phone-arrow-up-right"
description: "Alle Outbound-Assistenten für den authentifizierten Benutzer abrufen"
---

Dieser Endpunkt gibt eine Liste aller Outbound-Assistenten zurück, die dem authentifizierten Benutzer gehören. Outbound-Assistenten werden für das Tätigen von Anrufen an Leads und Interessenten verwendet.

### Antwort-Felder

Die API gibt ein direktes Array von Outbound-Assistenten zurück:

<ResponseField name="array" type="array">
  Array der Outbound-Assistenten

  <Expandable title="Assistenten-Eigenschaften">
    <ResponseField name="id" type="integer">
      Die eindeutige Kennung des Assistenten
    </ResponseField>

    <ResponseField name="user_id" type="integer">
      Die ID des Benutzers, dem dieser Assistent gehört
    </ResponseField>

    <ResponseField name="name" type="string">
      Der Name des Assistenten
    </ResponseField>

    <ResponseField name="type" type="string">
      Der Typ des Assistenten (wird immer "outbound" für diesen Endpunkt sein)
    </ResponseField>

    <ResponseField name="mode" type="string">
      Der Engine-Modus (pipeline oder multimodal)
    </ResponseField>

    <ResponseField name="status" type="string">
      Der aktuelle Status des Assistenten
    </ResponseField>

    <ResponseField name="voice_id" type="integer">
      Die ID der vom Assistenten verwendeten Stimme
    </ResponseField>

    <ResponseField name="language_id" type="integer">
      Die ID der vom Assistenten verwendeten Sprache
    </ResponseField>

    <ResponseField name="timezone" type="string">
      Die Zeitzoneneinstellung für den Assistenten
    </ResponseField>

    <ResponseField name="initial_message" type="string">
      Die erste Nachricht, die der Assistent sprechen wird
    </ResponseField>

    <ResponseField name="system_prompt" type="string">
      Der System-Prompt, der das Verhalten des Assistenten definiert
    </ResponseField>

    <ResponseField name="variables" type="object">
      Benutzerdefinierte Variablen für den Assistenten
    </ResponseField>

    <ResponseField name="webhook_url" type="string">
      Die Webhook-URL für Post-Call-Benachrichtigungen
    </ResponseField>

    <ResponseField name="is_webhook_active" type="boolean">
      Ob Webhook-Benachrichtigungen aktiviert sind
    </ResponseField>

    <ResponseField name="created_at" type="string">
      Datum und Uhrzeit der Erstellung des Assistenten
    </ResponseField>

    <ResponseField name="updated_at" type="string">
      Datum und Uhrzeit der letzten Aktualisierung des Assistenten
    </ResponseField>
  </Expandable>
</ResponseField>

### Hinweise

- Dieser Endpunkt gibt nur Assistenten mit dem Typ "outbound" zurück
- Alle Assistenten, die dem authentifizierten Benutzer gehören, werden zurückgegeben (keine Paginierung)
- Verwenden Sie diese Assistenten mit dem Make Call Endpunkt, um ausgehende Anrufe zu initiieren
- Outbound-Assistenten können für Verkaufsgespräche, Follow-ups, Umfragen und andere proaktive Anrufszenarien verwendet werden

<RequestExample>

```bash Outbound-Assistenten abrufen
curl -H "Authorization: Bearer YOUR_API_TOKEN" \
  "https://app.famulor.de/api/user/assistants/outbound"
```

</RequestExample>
