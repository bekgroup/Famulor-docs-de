---
title: "Benutzerinformationen abrufen"
api: "GET https://app.famulor.de/api/user/me"
icon: "user"
description: "Ruft die Informationen des authentifizierten Benutzers ab"
---

Dieser Endpunkt gibt die Profilinformationen des aktuell authentifizierten Benutzers zurück, einschließlich des Kontoguthabens.

### Antwort-Felder

<ResponseField name="name" type="string">
  Der vollständige Name des Benutzers
</ResponseField>

<ResponseField name="email" type="string">
  Die E-Mail-Adresse des Benutzers
</ResponseField>

<ResponseField name="total_balance" type="number">
  Das aktuelle Gesamtguthaben des Benutzerkontos
</ResponseField>

### Anwendungsfälle

Dieser Endpunkt ist nützlich für:

- **Authentifizierung überprüfen** - Bestätigen, dass der API-Schlüssel gültig ist und dem erwarteten Benutzer gehört
- **Kontoguthaben prüfen** - Verfügbare Credits überwachen, bevor API-Aufrufe getätigt werden
- **Benutzerinformationen anzeigen** - Den aktuellen Benutzer in der Benutzeroberfläche Ihrer Anwendung anzeigen

<RequestExample>

```bash Benutzerinformationen abrufen
curl -X GET "https://app.famulor.de/api/user/me" \
     -H "Authorization: Bearer YOUR_API_KEY" \
     -H "Content-Type: application/json"
```

</RequestExample>

<ResponseExample>

```json 200 Response
{
  "name": "Your Name",
  "email": "example@example.de",
  "total_balance": 100.67447094
}
```

</ResponseExample>
