---
title: "Dokument aktualisieren"
api: "PUT https://app.famulor.de/api/user/knowledgebases/{knowledgebaseId}/documents/{documentId}"
icon: "pencil"
description: "Aktualisiert den Namen oder die Beschreibung eines Dokuments"
---

Dieser Endpunkt aktualisiert die Metadaten eines bestehenden Dokuments. Sie können nur den Namen und die Beschreibung aktualisieren - der Dokumenttyp und der Inhalt können nicht geändert werden.

### Pfad-Parameter

<ParamField path="knowledgebaseId" type="integer" required>
  Die eindeutige Kennung der Wissensdatenbank
</ParamField>

<ParamField path="documentId" type="integer" required>
  Die eindeutige Kennung des zu aktualisierenden Dokuments
</ParamField>

### Request Body

<ParamField body="name" type="string">
  Der neue Name für das Dokument (max. 255 Zeichen)
</ParamField>

<ParamField body="description" type="string">
  Die neue Beschreibung für das Dokument (max. 255 Zeichen)
</ParamField>

### Antwort-Felder

<ResponseField name="message" type="string">
  Erfolgsmeldung
</ResponseField>

<ResponseField name="data" type="object">
  Das aktualisierte Dokument-Objekt
  
  <Expandable title="Daten-Eigenschaften">
    <ResponseField name="id" type="integer">
      Die eindeutige Kennung des Dokuments
    </ResponseField>

    <ResponseField name="name" type="string">
      Der Name des Dokuments
    </ResponseField>

    <ResponseField name="description" type="string">
      Beschreibung des Dokuments
    </ResponseField>

    <ResponseField name="type" type="string">
      Dokumenttyp
    </ResponseField>

    <ResponseField name="type_label" type="string">
      Menschenlesbares Typ-Label
    </ResponseField>

    <ResponseField name="status" type="string">
      Verarbeitungsstatus
    </ResponseField>

    <ResponseField name="status_label" type="string">
      Menschenlesbares Status-Label
    </ResponseField>

    <ResponseField name="updated_at" type="string">
      ISO 8601 Zeitstempel der letzten Aktualisierung
    </ResponseField>
  </Expandable>
</ResponseField>

<Note>
  Um den Inhalt eines Dokuments zu aktualisieren, müssen Sie das bestehende Dokument löschen und ein neues mit dem aktualisierten Inhalt erstellen.
</Note>

<RequestExample>

```bash Dokument aktualisieren
curl -X PUT "https://app.famulor.de/api/user/knowledgebases/1/documents/1" \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Aktualisiertes Produkthandbuch",
    "description": "Überarbeitetes Benutzerhandbuch mit neuen Funktionen"
  }'
```

</RequestExample>

<ResponseExample>


</ResponseExample>
