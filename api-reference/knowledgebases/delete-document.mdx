---
title: "Dokument löschen"
api: "DELETE https://app.famulor.de/api/user/knowledgebases/{knowledgebaseId}/documents/{documentId}"
icon: "trash"
description: "Löscht ein Dokument aus einer Wissensdatenbank"
---

Dieser Endpunkt löscht ein Dokument dauerhaft aus einer Wissensdatenbank. Die Vektor-Einbettungen des Dokuments werden ebenfalls aus der Vektordatenbank entfernt.

### Pfad-Parameter

<ParamField path="knowledgebaseId" type="integer" required>
  Die eindeutige Kennung der Wissensdatenbank
</ParamField>

<ParamField path="documentId" type="integer" required>
  Die eindeutige Kennung des zu löschenden Dokuments
</ParamField>

### Antwort-Felder

<ResponseField name="message" type="string">
  Erfolgsmeldung
</ResponseField>

### Was wird gelöscht

Wenn Sie ein Dokument löschen:

* **Dokumentdatensatz** wird aus der Datenbank entfernt
* **Vektor-Einbettungen** werden aus der Vektordatenbank gelöscht
* **Verarbeitete Inhalte** werden aus externen Dokumentenspeichern entfernt
* **Status der übergeordneten Wissensdatenbank** wird automatisch aktualisiert

<Warning>
  Diese Aktion ist unwiderruflich. Das Dokument und alle seine verarbeiteten Inhalte werden dauerhaft gelöscht.
</Warning>

<Note>
  Wenn dies das letzte Dokument in der Wissensdatenbank war, ändert sich der Status der Wissensdatenbank zu `empty`.
</Note>

<RequestExample>

```bash Dokument löschen
curl -X DELETE "https://app.famulor.de/api/user/knowledgebases/1/documents/1" \
     -H "Authorization: Bearer YOUR_API_KEY" \
     -H "Content-Type: application/json"
```

</RequestExample>

<ResponseExample>

```json 200 Success
{
  "message": "Document deleted successfully."
}
```

```json 404 Knowledgebase Not Found
{
  "error": "Knowledgebase not found."
}
```

```json 404 Document Not Found
{
  "error": "Document not found."
}
```

```json 500 Error
{
  "error": "Failed to delete document. Please try again."
}
```

</ResponseExample>
