---
title: "Mid-Call-Tool aktualisieren"
api: "PUT /user/tools/{id}"
description: "Ein bestehendes Mid-Call-Tool aktualisieren"
---

Dieser Endpunkt ermöglicht es Ihnen, ein bestehendes Mid-Call-Tool zu aktualisieren. Alle Felder sind optional - geben Sie nur die Felder an, die Sie aktualisieren möchten.

### Pfad-Parameter

<ParamField path="id" type="integer" required>
  Die eindeutige Kennung des zu aktualisierenden Tools
</ParamField>

### Body-Parameter

<ParamField body="name" type="string" optional>
  Tool-Name - darf nur Kleinbuchstaben und Unterstriche enthalten und muss mit einem Buchstaben beginnen
</ParamField>

<ParamField body="description" type="string" optional>
  Detaillierte Erklärung, wann und wie die KI dieses Tool verwenden soll (max. 255 Zeichen)
</ParamField>

<ParamField body="endpoint" type="string" optional>
  Gültige URL des aufzurufenden API-Endpunkts
</ParamField>

<ParamField body="method" type="string" optional>
  HTTP-Methode: `GET`, `POST`, `PUT`, `PATCH` oder `DELETE`
</ParamField>

<ParamField body="timeout" type="integer" optional>
  Request-Timeout in Sekunden (1-30)
</ParamField>

<ParamField body="headers" type="array" optional>
  HTTP-Header, die mit der Anfrage gesendet werden sollen (ersetzt bestehende Header)
  <Expandable title="Header-Eigenschaften">
    <ParamField body="name" type="string" required>
      Header-Name
    </ParamField>
    <ParamField body="value" type="string" required>
      Header-Wert
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="schema" type="array" optional>
  Parameter-Schema (ersetzt bestehendes Schema)
  <Expandable title="Schema-Eigenschaften">
    <ParamField body="name" type="string" required>
      Parameter-Name (2-32 Zeichen)
    </ParamField>
    <ParamField body="type" type="string" required>
      Parameter-Typ: `string`, `number` oder `boolean`
    </ParamField>
    <ParamField body="description" type="string" required>
      Parameter-Beschreibung (3-255 Zeichen)
    </ParamField>
  </Expandable>
</ParamField>

<ResponseField name="message" type="string">

<ResponseField name="data" type="object">

### Tool-Zuweisungen verwalten

Um dieses Tool zu Assistenten hinzuzufügen oder zu entfernen, verwenden Sie die Assistenten-API:

- **[Assistent erstellen](/api-reference/assistants/create)** - Verwenden Sie den `tool_ids`-Parameter, um Tools beim Erstellen eines Assistenten zuzuweisen
- **[Assistent aktualisieren](/api-reference/assistants/update)** - Verwenden Sie den `tool_ids`-Parameter, um zu verwalten, welche Tools einem Assistenten zugewiesen sind
